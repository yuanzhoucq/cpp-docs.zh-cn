# Array::Value 属性
检索当前数组的句柄。  
  
## 语法  
  
```  
  
property Array^ Value;  
```  
  
## 返回值  
 当前数组的句柄。  
  
## 要求  
 **标头：**vccorlib.h  
  
 **命名空间：**Platform
