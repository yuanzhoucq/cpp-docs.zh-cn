# <a name="commonly-added-member-functions"></a>经常添加的成员函数

如果对话框包含 "确定" 或 "取消" 之外的按钮，则需要在对话框类中编写消息处理程序成员函数，以响应其生成的控件通知消息。 有关示例，请参阅[随意绘制](../overview/visual-cpp-samples.md)的示例程序。 你还可以处理来自对话框中其他控件的控件通知消息。
