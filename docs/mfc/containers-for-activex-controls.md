# <a name="containers-for-activex-controls"></a>ActiveX 控件的容器

可以在其他应用程序中使用在 Visual C++ 中开发的 ActiveX 控件，前提是它们支持 ActiveX 控件包含。 许多 Microsoft 应用程序（从列出的版本开始）支持 ActiveX 控件包含。

>[!IMPORTANT]
> ActiveX 是一种不能用于新开发的旧技术。 有关取代 ActiveX 的新式技术的详细信息，请参阅[Activex 控件](activex-controls.md)。

> [!NOTE]
> 以下列表不是支持 ActiveX 控件的应用程序的完整列表，而是表示测试中最常使用的集：

支持 ActiveX 控件包含的应用程序包括：

- Internet Explorer 2.x 及更高版本

- Visual Basic 1.x 和更高版本

- Visual C++ 1.x 和更高版本

- Access 95 和更高版本

- Excel 97 及更高版本

- Word 97 及更高版本

- Access 97 和更高版本

- FrontPage 97 及更高版本

- PowerPoint 97 及更高版本

- Visual InterDev 97 及更高版本

下面是支持 ActiveX 控件包含的非 Microsoft 应用程序：

- PowerBuilder

- Delphi

- C + + 生成器

- 用于 Netscape navigator 的 NCompass 插件
