# <a name="using-cricheditctrl"></a>使用 CRichEditCtrl

Rich edit 控件是一个窗口，用户可以在其中输入和编辑文本。 可以为文本分配字符和段落格式，还可以包括嵌入的 OLE 对象。 Rich edit 控件由[CRichEditCtrl](../mfc/reference/cricheditctrl-class.md)类在 MFC 中表示。
