## 备注  
 使用 `CMFCVisualManagerOffice2007` 更改应用程序的视觉外观类似于Microsoft Office 2007。  此视觉管理器需要将样式，请在才能使用。  在通过此视觉管理器到 `CMFCVisualManager::SetDefaultManager`之前，必须调用静态函数 `CMFCVisualManagerOffice2007::SetStyle`。
