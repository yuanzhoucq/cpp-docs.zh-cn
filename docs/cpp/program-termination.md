# <a name="program-termination"></a>程序终止
在 C++ 中，可通过多种方式退出程序：  
  
-   调用[退出](../cpp/exit-function.md)函数。  
  
-   调用[中止](../cpp/abort-function.md)函数。  
  
-   执行[返回](../cpp/return-statement-in-program-termination-cpp.md)语句从`main`。
