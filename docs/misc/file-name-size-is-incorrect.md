# File &lt;name&gt; size is incorrect.
该文件包含的字节比预期的少，可能因为文件被截断或已损坏。  
  
### 更正此错误  
  
1.  关闭 Visual Studio，然后将其重新启动。  
  
2.  如果错误继续存在，请重新安装 Visual Studio 以修复该文件。
