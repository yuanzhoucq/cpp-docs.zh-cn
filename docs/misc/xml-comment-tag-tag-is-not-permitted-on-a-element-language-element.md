# “&lt;element&gt;”语言元素中不允许出现 XML 注释标记“&lt;tag&gt;”
“\<element\>”元素中不允许出现 XML 注释标记 \<tag\>。 将忽略 XML 注释。  
  
 应用 XML 注释标记的语言元素不支持该标记。  
  
 **错误 ID：**BC42306  
  
### 更正此错误  
  
-   删除不受支持的 XML 注释标记。
