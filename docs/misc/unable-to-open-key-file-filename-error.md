# 无法打开密钥文件“&lt;filename&gt;”：&lt;error&gt;
无法打开为此项目指定的密钥文件。  
  
 **错误 ID：**BC31025  
  
### 更正此错误  
  
-   验证该密钥文件是否存在，并且未被另一个应用程序锁定。
