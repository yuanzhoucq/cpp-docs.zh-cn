# 在条件编译表达式中不允许有非内部的类型名称
条件编译表达式引用非内部类型，但仅允许使用内部类型。  
  
 有关 [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] 中内部类型的详细信息，请参阅 [数据类型摘要](../Topic/Data%20Types%20Summary%20\(Visual%20Basic\).md)。  
  
 **错误 ID：**BC31426  
  
### 更正此错误  
  
-   将非内部类型替换为内部类型。
