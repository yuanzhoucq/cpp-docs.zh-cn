# Visual Studio requires Windows 2000 or above.
Visual Studio 要求 Windows NT 4.0（内部版本 1381，带有 Service Pack 3）或 Windows 95（内部版本 708）或更高版本。  
  
### 更正此错误  
  
1.  将 Windows NT 或 Windows 的当前版本升级到上面列出的版本。
