# “AddHandler”声明必须以匹配的“End AddHandler”结束
`AddHandler` 声明必须以 `End AddHandler` 语句结束。  
  
 **错误 ID：**BC31115  
  
### 更正此错误  
  
-   检查 `AddHandler` 声明是否以 `End AddHandler` 语句结束。
