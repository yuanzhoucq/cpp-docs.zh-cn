# 无法向未声明为“Optional”的参数提供默认值
调用过程时必须提供所需的参数，并且这些参数不接受声明中的默认值。  
  
 **错误 ID：**BC32024  
  
### 更正此错误  
  
-   将参数声明为 `Optional` 或删除默认值。
