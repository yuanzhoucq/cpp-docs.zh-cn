# 指定的注册表路径开头不是有效的配置单元名称
指定的注册表路径开头不是有效的配置单元名称。 有效的配置单元名称为：  
  
-   `HKEY_CLASSES_ROOT`  
  
-   `HKEY_CURRENT_CONFIG`  
  
-   `HKEY_CURRENT_USER`  
  
-   `HKEY_DYNDATA`  
  
-   `HKEY_LOCAL_MACHINE`  
  
-   `HKEY_PERFORMANCE_DATA`  
  
-   `HKEY_USERS`  
  
### 更正此错误  
  
-   验证当前提供的配置单元名称是否有效。
