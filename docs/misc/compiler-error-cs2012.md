# 编译器错误 CS2012
无法打开“file”进行写入  
  
 当使用 [\/bugreport:](../Topic/-bugreport%20\(C%23%20Compiler%20Options\).md)`file` 编译器选项时，未能打开文件进行写入。 请确保指定的文件名有效，并且该文件不是只读文件。
