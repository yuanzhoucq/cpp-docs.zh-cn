# 枚举必须声明为整型。
可以在枚举中使用的唯一有效类型是 `SByte`、`Byte`、`Short`、`UShort`、 `Integer`、`UInteger`、`Long` 和 `ULong`。 不能使用其他数据类型。  
  
 **错误 ID：**BC30650  
  
### 更正此错误  
  
-   指定 `SByte`、`Byte`、`Short`、`UShort`、`Integer`、`UInteger`、`Long` 或 `ULong` 的数据类型。
