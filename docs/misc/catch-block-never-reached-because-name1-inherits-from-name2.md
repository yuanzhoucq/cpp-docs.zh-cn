# 从未到达“Catch”块，因为“&lt;name1&gt;”从“&lt;name2&gt;”继承。
从未到达代码中的 `Catch` 块。  
  
 默认情况下，此消息是一个警告。 有关隐藏警告或将警告视为错误的详细信息，请参见 [在 Visual Basic 中配置警告](../Topic/Configuring%20Warnings%20in%20Visual%20Basic.md)。  
  
 **错误 ID：**BC42029  
  
### 更正此错误  
  
-   删除无关的 `Catch` 块。
