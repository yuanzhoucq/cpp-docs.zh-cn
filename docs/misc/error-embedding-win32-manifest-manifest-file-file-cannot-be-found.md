# 嵌入 Win32 清单时出错：找不到清单文件 &lt;file&gt;
找不到为 `/win32manifest` 编译器选项指定的文件。  
  
 **错误 ID：**BC31192  
  
### 更正此错误  
  
1.  为 `/win32manifest` 编译器选项提供有效的文件名和路径。
