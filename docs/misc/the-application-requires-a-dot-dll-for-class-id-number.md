# The application requires a .dll for class ID &lt;number&gt;.
Visual Studio 未能找到您指定的类所需的 DLL。  这是操作系统错误。  
  
### 更正此错误  
  
1.  重新安装 Visual Studio 或受影响的应用程序并重试。
