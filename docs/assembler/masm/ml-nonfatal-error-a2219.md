# <a name="ml-nonfatal-error-a2219"></a>ML 非致命错误 A2219
**错误中的偏移量的对齐方式展开代码**  
  
 操作数[。ALLOCSTACK](../../assembler/masm/dot-allocstack.md)和[。SAVEREG](../../assembler/masm/dot-savereg.md)必须是 8 的倍数。  操作数[。SAVEXMM128](../../assembler/masm/dot-savexmm128.md)和[。SETFRAME](../../assembler/masm/dot-setframe.md)必须是 16 的倍数。
