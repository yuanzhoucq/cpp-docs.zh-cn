# <a name="union"></a>UNION
声明的一个或多个数据类型的联合。 *Fielddeclarations*必须是有效的数据定义。 省略[结束](../../assembler/masm/ends-masm.md)*名称*上的标签嵌套**联合**定义。  
  
## <a name="syntax"></a>语法  
  
```  
  
      name   
      UNION [[alignment]] [[, NONUNIQUE]]  
   fielddeclarations  
[[name]] ENDS  
```
