# <a name="union"></a>UNION

声明一个或多个数据类型的联合。 *字段声明*必须是有效的数据定义。 省略[结束](../../assembler/masm/ends-masm.md)*名称*标签上的嵌套**联合**定义。

## <a name="syntax"></a>语法

> *名称*UNION [[*对齐*]] [[，唯一，则]]<br/>
> *field-declarations*<br/>
> [[*name*]] ENDS
