# <a name="sqword"></a>SQWORD
分配和 （可选） 为每个初始化 8 字节有符号的存储`initializer`。 此外可以用作类型说明符类型是合法的任何位置。  
  
## <a name="syntax"></a>语法  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
