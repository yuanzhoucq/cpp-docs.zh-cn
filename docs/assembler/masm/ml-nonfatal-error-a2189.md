# ML Nonfatal Error A2189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**与段落对齐方式的无效组合**  
  
 **对齐** 或 `EVEN` 指令指定的对齐小于当前段落对齐方式。根据指定的 **段** 指令。
