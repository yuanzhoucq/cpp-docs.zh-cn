# <a name="sbyte"></a>SBYTE

分配并选择性地为每个*初始值设定项*初始化一个有符号字节的存储。 在类型合法的任何位置，还可以用作类型说明符。

## <a name="syntax"></a>语法

> *命名*  **SBYTE** *初始值*⟦ __，__ *初始值设定项*.。。⟧
