# operator SHORT
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

设置标记类型为 short。  *标记的* 所有跳转必须是短 \(在范围 –从跳转越命令的 128 到 \+127 个字节中 *标记*\)。  
  
## 语法  
  
```  
  
SHORT label  
```
