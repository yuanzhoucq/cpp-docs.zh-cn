# <a name="operator-highword"></a>运算符 HIGHWORD
返回的低 32 位的高 16 位*表达式*。 MASM 表达式是 64 位值。  
  
## <a name="syntax"></a>语法  
  
```  
  
HIGHWORD expression  
```
