# operator HIGHWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回低 32 位的高 16 位 *表达式*。  MASM 表达式为 64 位值。  
  
## 语法  
  
```  
  
HIGHWORD expression  
```
