# REAL4
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

分配和 \(可选\) 初始化每个 *初始值设定项的*一个单精度 \(4 字节\) 浮点数字。  
  
## 语法  
  
```  
  
name REAL4 initializer [[, initializer]]...  
```
