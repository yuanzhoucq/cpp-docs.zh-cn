# .LISTMACRO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

启动 " 列出生成代码或数据的宏展开语句。  
  
## 语法  
  
```  
  
.LISTMACRO  
  
```  
  
## 备注  
 这是默认值。  和 [.XALL](../../assembler/masm/dot-xall.md)相同。
