# ML Nonfatal Error A2047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**空 \(空\) 字符串**  
  
 字符串包含用于分隔对引号和不字符内。  
  
 为使字符串有效，它必须包含 1\-255 字符。
