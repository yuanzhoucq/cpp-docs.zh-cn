# ECHO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

显示消息传送到标准输出设备 \(默认情况下，屏幕\)。  
  
## 语法  
  
```  
  
ECHO   
message  
  
```  
  
## 备注  
 和 [%OUT](../../assembler/masm/percent-out.md)相同。
