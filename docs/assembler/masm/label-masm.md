# <a name="label-masm"></a>LABEL (MASM)
通过分配的当前的位置计数器值创建一个新标签与给定`type`到*名称*。  
  
## <a name="syntax"></a>语法  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```
