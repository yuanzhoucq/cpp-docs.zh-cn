# <a name="textequ"></a>TEXTEQU
将分配*textitem*到*名称*。 *Textitem*可以是文本字符串，前面是一个常数`%`，或通过宏函数返回的字符串。  
  
## <a name="syntax"></a>语法  
  
```  
  
name TEXTEQU [[textitem]]  
```
