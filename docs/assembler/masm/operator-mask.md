# operator MASK
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

返回在 *recordfieldname* 或 *记录的* 位设置，以及其他 BITS 清除的位掩码。  
  
## 语法  
  
```  
  
MASK {recordfieldname | record}  
```
