# <a name="comment-masm"></a>COMMENT (MASM)
将所有*文本*之间或为一个注释分隔符与位于同一行上。  
  
## <a name="syntax"></a>语法  
  
```  
  
      COMMENT delimiter [[text]]   
[[text]]   
[[text]] delimiter [[text]]  
```
