# <a name="align-masm"></a>ALIGN (MASM)
根据区域性对齐下一步的变量或上一个字节的倍数的指令`number`。  
  
## <a name="syntax"></a>语法  
  
```  
  
ALIGN [[  
number  
]]  
  
```
