# <a name="datasize"></a>@DataSize
0 表示**微小**，**小**，**中等**，和**平面**模型，1 表示**COMPACT**和**大型**模型和 2**非常巨大**模型 （数字相当）。  
  
## <a name="syntax"></a>语法  
  
```  
  
@DataSize  
  
```
