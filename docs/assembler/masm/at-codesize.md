# <a name="codesize"></a>@CodeSize
0 表示**微小**，**小**， **COMPACT**，和**平面**模型，并为 1**中等**， **大型**，和**非常巨大**（数字等同） 的模型。  
  
## <a name="syntax"></a>语法  
  
```  
  
@CodeSize  
  
```
