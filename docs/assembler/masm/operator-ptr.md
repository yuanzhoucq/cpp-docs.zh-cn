# <a name="operator-ptr"></a>运算符 PTR

The first operator forces the *expression* to be treated as having the specified *type*. The second operator specifies a pointer to *type*.

## <a name="syntax"></a>语法

> *type* **PTR** *expression*
>
> \[*distance*] **PTR** *type*
