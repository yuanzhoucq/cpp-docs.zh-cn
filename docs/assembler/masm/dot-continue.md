# <a name="continue-32-bit-masm"></a>.CONTINUE （32位 MASM）

生成要跳转到顶部的代码[。WHILE](../../assembler/masm/dot-while.md)或[。](../../assembler/masm/dot-repeat.md)如果 `condition` 为 true，则重复块。 （仅限32位 MASM。）
