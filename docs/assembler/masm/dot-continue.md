# <a name="continue"></a>.CONTINUE
生成代码，以跳转到顶部[。虽然](../../assembler/masm/dot-while.md)或[。重复](../../assembler/masm/dot-repeat.md)阻止如果`condition`为 true。  
  
## <a name="syntax"></a>语法  
  
```  
  
.CONTINUE [[.IF condition]]  
```
