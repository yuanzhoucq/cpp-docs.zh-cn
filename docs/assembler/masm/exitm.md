# <a name="exitm"></a>EXITM

终止当前的重复或宏块的扩展，并开始下一个语句块外的程序集。

## <a name="syntax"></a>语法

> EXITM [textitem]

## <a name="remarks"></a>备注

在宏函数中， *textitem*返回的值。
