# <a name="forc"></a>FORC

将标记将为在每个字符一次重复的块*字符串*，使用当前的字符替换*参数*上每次重复。

## <a name="syntax"></a>语法

> FORC*参数*， \<*字符串*><br/>
> *语句*<br/>
> ENDM
