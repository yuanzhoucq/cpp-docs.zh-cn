# <a name="break"></a>.BREAK
生成代码，以终止[。虽然](../../assembler/masm/dot-while.md)或[。重复](../../assembler/masm/dot-repeat.md)阻止如果*条件*为 true。  
  
## <a name="syntax"></a>语法  
  
```  
  
.BREAK [[.IF condition]]   
```
