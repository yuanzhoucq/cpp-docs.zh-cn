# <a name="exit-32-bit-masm"></a>.EXIT (32-bit MASM)

Generates termination code. (32-bit MASM only.)

## <a name="syntax"></a>语法

> **.EXIT** ⟦*expression*⟧

## <a name="remarks"></a>备注

Returns optional *expression* to shell.
