# <a name="exit-32-bit-masm"></a>.EXIT （32位 MASM）

生成终止代码。 （仅限32位 MASM。）

## <a name="syntax"></a>语法

> **.EXIT** ⟦*expression*⟧

## <a name="remarks"></a>备注

返回 shell 的可选*表达式*。
