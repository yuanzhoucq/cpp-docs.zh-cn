# <a name="struct-masm"></a>STRUCT (MASM)

声明具有指定*字段声明*的结构类型。 每个字段都必须是有效的数据定义。 与[STRUC](../../assembler/masm/struc.md)相同。

## <a name="syntax"></a>语法

> *name* **STRUCT** ⟦*对齐*⟧⟦ __，__ 非**唯一**⟧*字段声明*\
> *名称***结束**
