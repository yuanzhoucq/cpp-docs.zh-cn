# <a name="charsformat-class"></a>chars_format 类

基元数值转换为浮点格式。

## <a name="syntax"></a>语法

```cpp
enum class chars_format {
    scientific = unspecified ,
    fixed = unspecified ,
    hex = unspecified ,
    general = fixed | scientific
};
```
