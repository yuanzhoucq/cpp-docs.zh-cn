# <a name="trytolockt-structure"></a>try_to_lock_t 结构

表示一个定义[try_to_lock](../standard-library/mutex-functions.md#try_to_lock)对象的**结构**。 用于选择 [unique_lock](../standard-library/unique-lock-class.md) 的重载构造函数之一。

## <a name="syntax"></a>语法

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>要求

**标头:** \<mutex >

**命名空间：** std
