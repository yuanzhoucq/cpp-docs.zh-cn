# <a name="debug"></a>_DEBUG
指定 /MTd 或 /MDd 选项时，编译器将定义 `_DEBUG`。 这些选项可指定 C 运行时库的调试版本。  
  
 有关详细信息，请参阅 [CRT 调试方法](/visualstudio/debugger/crt-debugging-techniques)。
