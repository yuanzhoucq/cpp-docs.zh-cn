# <a name="andeq"></a>and_eq

&= 运算符的替代项。

## <a name="syntax"></a>语法

```C

#define and_eq &=

```

## <a name="remarks"></a>备注

该宏产生运算符 &=。

## <a name="example"></a>示例

```cpp
// iso646_and_eq.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a &= b;
   cout << result << endl;

   result= a and_eq b;
   cout << result << endl;
}
```

```Output
2
2
```

## <a name="requirements"></a>要求

**标头：** \<iso646.h>
