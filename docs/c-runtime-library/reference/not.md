# <a name="not"></a>not

! 的替代项 运算符。

## <a name="syntax"></a>语法

```C

#define not !
```

## <a name="remarks"></a>备注

该宏产生运算符 !。

## <a name="example"></a>示例

```cpp
// iso646_not.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 0;

   if (!a)
      cout << "a is zero" << endl;

   if (not(a))
      cout << "a is zero" << endl;
}
```

```Output
a is zero
a is zero
```

## <a name="requirements"></a>要求

**标头：** \<iso646.h>
