# <a name="_getmbcp"></a>_getmbcp

检索当前代码页。

## <a name="syntax"></a>语法

```C
int _getmbcp( void );
```

## <a name="return-value"></a>返回值

返回当前多字节代码页。 返回值为 0 表示单字节代码页正在使用中。

## <a name="remarks"></a>备注

默认情况下，此函数的全局状态范围为应用程序。 要更改此情况，请参阅[CRT 中的全局状态](../global-state.md)。
