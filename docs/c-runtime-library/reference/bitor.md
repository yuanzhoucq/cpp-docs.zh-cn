# <a name="bitor"></a>bitor
&#124; 运算符的替代项。  
  
## <a name="syntax"></a>语法  
  
```  
  
#define bitor |  
  
```  
  
## <a name="remarks"></a>备注  
 该宏产生运算符 &#124;。  
  
## <a name="example"></a>示例  
  
```  
// iso646_bitor.cpp  
// compile with: /EHsc  
#include <iostream>  
#include <iso646.h>  
  
int main( )  
{  
   using namespace std;  
   int a = 1, b = 2, result;  
  
   result = a | b;  
   cout << result << endl;  
  
   result= a bitor b;  
   cout << result << endl;  
}  
```  
  
```Output  
3  
3  
```  
  
## <a name="requirements"></a>要求  
 **标头：** \<iso646.h>
