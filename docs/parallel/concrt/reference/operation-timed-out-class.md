# <a name="operationtimedout-class"></a>operation_timed_out 类
此类描述操作超时时引发的异常。  
  
## <a name="syntax"></a>语法  
  
```
class operation_timed_out : public std::exception;
```

 构造 `operation_timed_out` 对象。  
  
```
explicit _CRTIMP operation_timed_out(_In_z_ const char* _Message) throw();
```  
