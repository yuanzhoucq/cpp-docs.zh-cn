# <a name="work-with-odbc-and-drivers"></a>使用 ODBC 和驱动程序

ODBC 使你能够编写不依赖于一个数据库管理系统（DBMS）的应用程序。 可以编写一个应用程序，该应用程序可在多个 Dbms 上操作，用户具有正确的 DBMS 特定 ODBC 驱动程序。
