# <a name="cdynamicaccessorclose"></a>CDynamicAccessor::Close
取消绑定所有列，释放分配的内存，并释放[IAccessor](https://msdn.microsoft.com/en-us/library/ms719672.aspx)类中的接口指针。  
  
## <a name="syntax"></a>语法  
  
```cpp
void Close() throw();  
  
```  
  
## <a name="requirements"></a>要求  
 **标头:** atldbcli.h
