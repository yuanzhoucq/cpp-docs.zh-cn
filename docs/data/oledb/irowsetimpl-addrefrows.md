# <a name="irowsetimpladdrefrows"></a>IRowsetImpl::AddRefRows
将引用计数添加到现有行句柄。  
  
## <a name="syntax"></a>语法  
  
```  
  
      STDMETHOD( AddRefRows )(  
   DBCOUNTITEM cRows,  
   const HROW rghRows[],  
   DBREFCOUNT rgRefCounts[],  
   DBROWSTATUS rgRowStatus[]   
);  
```  
  
#### <a name="parameters"></a>参数
