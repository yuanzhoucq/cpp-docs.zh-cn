# <a name="carrayrowsetcarrayrowset"></a>CArrayRowset::CArrayRowset

## <a name="syntax"></a>语法  
  
```cpp
      CArrayRowset(int nMax = 100000);  
```  
  
#### <a name="parameters"></a>参数  
 `nMax`  
 [in] 行集中的最大行数。
