# <a name="cbookmarkoperator-"></a>CBookmark::operator =
将一个 `CBookmark` 对象分配给其他对象。  
  
## <a name="syntax"></a>语法  
  
```  
  
      CBookmark& operator =(   
   const CBookmark& bookmark    
) throw( );  
```  
  
## <a name="remarks"></a>备注  
 仅在需要此运算符**CBookmark\<0 >**。

 **标头:** atldbcli.h
