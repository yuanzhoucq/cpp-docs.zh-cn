# <a name="beginpropertyset"></a>BEGIN_PROPERTY_SET
在属性中设置属性的开头的标记集映射。  
  
## <a name="syntax"></a>语法  
  
```cpp
BEGIN_PROPERTY_SET(guid)  
  
```  
  
#### <a name="parameters"></a>参数  
 `guid`  
 [in]属性的 GUID 中。  
  
## <a name="example"></a>示例  
 请参阅 [BEGIN_PROPSET_MAP](../../data/oledb/begin-propset-map.md)。  
  
## <a name="requirements"></a>要求  
 **标头：** atldb.h
