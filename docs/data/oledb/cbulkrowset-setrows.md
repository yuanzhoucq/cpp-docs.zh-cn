# <a name="cbulkrowsetsetrows"></a>CBulkRowset::SetRows
设置每个调用检索的行句柄数量。  
  
## <a name="syntax"></a>语法  
  
```  
  
      void SetRows(  
   DBROWCOUNT nRows   
) throw( );  
```  
  
#### <a name="parameters"></a>参数  
 `nRows`  
 [in] 行集的新大小（行数）。  
  
## <a name="remarks"></a>备注  
 如果调用此函数，则它必须在行集之前打开。

 **标头:** atldbcli.h
