# <a name="crowsetgetdatahere"></a>CRowset::GetDataHere
从当前行中检索数据并将其放到指定的缓冲区。  
  
## <a name="syntax"></a>语法  
  
```  
  
      HRESULT GetDataHere(   
   int nAccessor,   
   void* pBuffer    
) throw( );  
```  
  
#### <a name="parameters"></a>参数  
 `nAccessor`  
 [in]访问器可用于访问的数据索引号。  
  
 `pBuffer`  
 [out]到来放置当前记录的数据缓冲区。  
  
## <a name="return-value"></a>返回值  
 一个标准 `HRESULT`。
