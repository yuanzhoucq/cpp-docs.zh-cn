# <a name="cdbpropsetoperator-"></a>CDBPropSet::operator =
将一个属性集的内容分配给另一属性集。  
  
## <a name="syntax"></a>语法  
  
```cpp
      CDBPropSet& operator =(CDBPropSet& propset) throw();  
```
