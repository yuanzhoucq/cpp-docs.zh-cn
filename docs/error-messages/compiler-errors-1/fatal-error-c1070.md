# <a name="fatal-error-c1070"></a>错误 C1070
文件“filename”中的 #if/#endif 对不匹配  
  
 `#if`、 `#ifdef`或 `#ifndef` 指令没有对应的 `#endif`。  
  
 下面的示例生成 C1070：  
  
```  
// C1070.cpp  
#define TEST  
  
#ifdef TEST  
  
#ifdef TEST  
#endif  
// C1070  
```  
  
 可能的解决方法：  
  
```  
// C1070b.cpp  
// compile with: /c  
#define TEST  
  
#ifdef TEST  
#endif  
  
#ifdef TEST  
#endif  
```
