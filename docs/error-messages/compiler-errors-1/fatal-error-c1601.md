# <a name="fatal-error-c1601"></a>错误 C1601

不支持的内联程序集操作码

此错误指示编译器.exe 文件，这可能是由于不完整的安装中不匹配。 例如，你可能已安装服务包，但不是 Processor Pack。 安装所有必需的产品。
