# <a name="compiler-error-c2451"></a>编译器错误 C2451
条件表达式 type 是类型的非法的  
  
 条件表达式的计算结果为整数类型。  
  
 下面的示例生成 C2451:  
  
```  
// C2451.cpp  
class B {};  
  
int main () {  
   B b1;  
   int i = 0;  
   if (b1)   // C2451  
   // try the following line instead  
   // if (i)  
      ;  
}  
```
