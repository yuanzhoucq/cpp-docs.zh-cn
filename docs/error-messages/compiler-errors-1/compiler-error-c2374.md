# <a name="compiler-error-c2374"></a>编译器错误 C2374
“identifier”：重定义；多次初始化  
  
 不止一次初始化了标识符。  
  
 下面的示例生成 C2374：  
  
```  
// C2374.cpp  
// compile with: /c  
int i = 0;  
int i = 1;   // C2374  
int j = 1;   // OK  
```
