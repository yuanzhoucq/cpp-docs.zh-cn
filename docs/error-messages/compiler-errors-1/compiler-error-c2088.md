# <a name="compiler-error-c2088"></a>编译器错误 C2088
operator： 非法的 class-key  
  
 没有为结构或联合定义运算符。 此错误是仅针对 C 代码有效。  
  
 下面的示例生成 C2088 三次：  
  
```  
// C2088.c  
struct S {  
   int m_i;   
} s;  
  
int main() {  
   int i = s * 1;   // C2088  
   struct S s2 = +s;   // C2088  
   s++;   // C2088  
}  
```
