# <a name="compiler-error-c2109"></a>编译器错误 C2109
下标需要数组或指针类型  
  
 该下标用了不是数组的变量上。  
  
 下面的示例生成 C2109:  
  
```  
// C2109.cpp  
int main() {  
   int a, b[10] = {0};  
   a[0] = 1;   // C2109  
   b[0] = 1;   // OK  
}  
```
