# <a name="compiler-error-c2428"></a>编译器错误 C2428
operation： 不允许对类型为 bool 的操作数  
  
 不能将递减运算符应用于类型的对象`bool`。  
  
 下面的示例生成 C2428:  
  
```  
// C2428.cpp  
void g(bool fFlag) {  
   --fFlag;   // C2428  
   fFlag--;   // C2428  
}  
```
