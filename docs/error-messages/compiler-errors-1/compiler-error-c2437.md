# <a name="compiler-error-c2437"></a>编译器错误 C2437

"identifier"：已初始化

对象只能初始化一次。

下面的示例生成 C2437：

```cpp
// C2437.cpp
// compile with: /c
class A {
public:
   A(int i) {}
};

class B : A {
   B() : A(1), A(2) {}   // C2437
   B() : A(1) {}   // OK
};
```
