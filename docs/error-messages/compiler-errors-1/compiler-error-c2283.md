# <a name="compiler-error-c2283"></a>编译器错误 C2283
“identifier”：未命名的结构上不允许使用纯说明符或抽象重写说明符  
  
 使用纯说明符声明了未命名类或结构的成员函数，这是不允许的。  
  
 以下示例生成 C2283：  
  
```  
// C2283.cpp  
// compile with: /c  
struct {  
   virtual void func() = 0;   // C2283  
};  
struct T {  
   virtual void func() = 0;   // OK  
};  
```
