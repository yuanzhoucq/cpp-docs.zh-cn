# <a name="compiler-error-c2324"></a>编译器错误 C2324
identifier： 意外右侧的 name  
  
 使用不正确的标识符调用析构函数。  
  
 下面的示例生成 C2324:  
  
```  
// C2324.cpp  
class A {};  
typedef A* pA_t;  
int i;  
  
int main() {  
   pA_t * ppa = new pA_t;  
   ppa->~i;   // C2324  
   ppa->~pA_t();   // OK  
}  
```
