# <a name="compiler-error-c2175"></a>编译器错误 C2175

"locale"：无效的区域设置

指定的区域设置无效。 有关受支持的区域设置，请参阅*运行库参考*中的[语言和国家/地区字符串](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)。
