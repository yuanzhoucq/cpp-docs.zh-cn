# <a name="compiler-error-c2233"></a>编译器错误 C2233

"identifier"：包含大小为零的数组的对象数组是非法的

数组中的每个对象都必须至少包含一个元素。

下面的示例生成 C2233：

```cpp
// C2233.cpp
// compile with: /c
class A {
   char somearray[1];
};

class B {
   char zeroarray[];
};

A array[100];   // OK
B array2[100];   // C2233
```
