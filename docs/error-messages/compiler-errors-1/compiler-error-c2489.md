# <a name="compiler-error-c2489"></a>编译器错误 C2489
identifier： 初始化函数范围中函数中不允许 naked 的自动或注册变量  
  
 有关详细信息，请参阅[裸](../../cpp/naked-cpp.md)。  
  
 下面的示例生成 C2489:  
  
```  
// C2489.cpp  
// processor: x86  
__declspec( naked ) int func() {  
   int i = 1;   // C2489  
   register int j = 1;   // C2489  
}  
```
