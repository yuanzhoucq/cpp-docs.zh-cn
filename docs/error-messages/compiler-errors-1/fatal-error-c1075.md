# 错误 C1075
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

左侧标记与文件结尾不匹配  
  
 编译器应在到达文件结尾之前匹配 *标记*。  
  
 导致此错误的原因可能是不匹配的括号、大括号或其他成对字符。
