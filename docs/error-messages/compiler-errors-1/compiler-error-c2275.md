# <a name="compiler-error-c2275"></a>编译器错误 C2275

identifier： 非法使用了此类型作为表达式

表达式使用了`->`运算符与`typedef`标识符。

下面的示例生成 C2275:

```
// C2275.cpp
typedef struct S {
    int mem;
} *S_t;
void func1( int *parm );
void func2() {
    func1( &S_t->mem );   // C2275, S_t is a typedef
}
```
