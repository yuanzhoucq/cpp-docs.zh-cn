# 编译器错误 C2275
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 将此类型用作表达式非法  
  
 表达式使用带有 `typedef` 标识符的 `->` 运算符。  
  
 下面的示例生成 C2275：  
  
```  
// C2275.cpp  
typedef struct S {  
    int mem;  
} *S_t;  
void func1( int *parm );  
void func2() {  
    func1( &S_t->mem );   // C2275, S_t is a typedef  
}  
```
