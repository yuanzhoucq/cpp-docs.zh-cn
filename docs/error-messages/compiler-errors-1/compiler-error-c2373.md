# <a name="compiler-error-c2373"></a>编译器错误 C2373
“identifier”：重定义；不同的类型修饰符  
  
 已使用不同的类型修饰符定义标识符。  
  
 下面的示例生成 C2373：  
  
```  
// C2373.h  
void __clrcall func( void );  
const int i = 20;  
```  
  
 然后：  
  
```  
// C2373.cpp  
// compile with: /c  
#include "C2373.h"  
extern void __cdecl func( void );   // C2373  
extern void __clrcall func( void );   // OK  
  
extern int i;   // C2373  
extern const int i;   // OK  
```
