# <a name="compiler-error-c2095"></a>编译器错误 C2095

function： 实参具有 void 类型： 参数 number

传递给函数的参数的类型为`void`，这不允许。 使用指向 void 的指针 ( `void *`) 相反。

`number`指示哪个参数为`void`。
