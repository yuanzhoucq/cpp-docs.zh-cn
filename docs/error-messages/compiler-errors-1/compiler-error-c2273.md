# <a name="compiler-error-c2273"></a>编译器错误 C2273
“type”: 位于“->”运算符右边时非法  
  
 类型的右操作数作为出现`->`运算符。  
  
 通过尝试访问用户定义类型转换可以导致此错误。 请在 -> 和 `operator` 之间使用关键字 `type`。  
  
 下面的示例生成 C2273:  
  
```  
// C2273.cpp  
struct MyClass {  
   operator int() {  
      return 0;  
   }  
};  
int main() {  
   MyClass * ClassPtr = new MyClass;  
   int i = ClassPtr->int();   // C2273  
   int j = ClassPtr-> operator int();   // OK  
}  
```
