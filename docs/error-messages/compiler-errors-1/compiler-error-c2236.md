# <a name="compiler-error-c2236"></a>编译器错误 C2236
意外的标记“identifier”。 你是否忘了“;”？  
  
 该标识符已经定义为类型，无法被用户定义类型重写。  
  
 下面的示例生成 C2236：  
  
```  
// C2236.cpp  
// compile with: /c  
int class A {};  // C2236 "int class" is unexpected  
int i;  
class B {};  
```
