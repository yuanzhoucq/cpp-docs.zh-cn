# <a name="compiler-error-c2010"></a>编译器错误 C2010
character： 意外宏形参表中  
  
 在宏定义的形参列表中错误地使用了该字符。 删除要解决该错误的字符。  
  
 下面的示例生成 C2010:  
  
```  
// C2010.cpp  
// compile with: /c  
#define mymacro(a|) (2*a)   // C2010  
#define mymacro(a) (2*a)   // OK  
```
