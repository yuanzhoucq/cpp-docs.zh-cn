# <a name="compiler-error-c2400"></a>编译器错误 C2400
context; 中的内联汇编程序语法错误找到 token  
  
 令牌在指定的上下文中导致语法错误。  
  
 下面的示例生成 C2400:  
  
```  
// C2400.cpp  
// processor: x86  
int main() {  
   __asm {  
      heh ax,bx;   // C2400, heh is not a valid x86 instruction  
      mov ax,bx;   // OK  
   }  
}  
```
