# <a name="fatal-error-c1109"></a>错误 C1109
在 DLL“dll”中找不到“入口点”  
  
 在编译器所需的 [延迟加载 DLL](../../build/reference/linker-support-for-delay-loaded-dlls.md) 中找不到入口点。
