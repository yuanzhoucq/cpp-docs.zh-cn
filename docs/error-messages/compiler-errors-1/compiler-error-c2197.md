# <a name="compiler-error-c2197"></a>编译器错误 C2197

“function”: 用于调用的参数太多

编译器检测到用于函数调用的参数太多或函数声明不正确。

下面的示例生成 C2197：

```
// C2197.c
// compile with: /Za /c
void func( int );
int main() {
   func( 1, 2 );   // C2197 two actual parameters
   func( 2 );   // OK
}
```
