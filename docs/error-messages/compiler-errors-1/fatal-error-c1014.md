# 错误 C1014
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

包含文件太多：深度 \= level  
  
 `#include` 指令的嵌套太深。 嵌套指令可以包括打开的文件。 包含该指令的源文件计为一个文件。
