# <a name="fatal-error-c1100"></a>错误 C1100

无法初始化 OLE： 系统错误消息

编译器无法初始化组件对象模型 (COM) 库。 请参阅[CoInitialize](/windows/desktop/api/objbase/nf-objbase-coinitialize)。
