# <a name="compiler-error-c2181"></a>编译器错误 C2181
没有匹配 if 的非法 else   
  
 每个 `else` 必须具有匹配的 `if`。  
  
 以下示例生成 C2181：  
  
```  
// C2181.cpp  
int main() {  
   int i = 0;  
   else   // C2181  
      i = 1;  
}  
```  
  
 可能的解决方法：  
  
```  
// C2181b.cpp  
int main() {  
   int i = 0;  
   if(i)  
      i = 0;  
   else  
      i = 1;  
}  
```
