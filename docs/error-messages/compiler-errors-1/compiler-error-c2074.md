# 编译器错误 C2074
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”：“class\-key”初始化需要大括号  
  
 指定的类、结构或联合初始值设定项两边没有大括号。
