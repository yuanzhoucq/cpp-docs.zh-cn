# <a name="compiler-error-c2121"></a>编译器错误 C2121

'#': 无效的字符： 可能是宏扩展的结果

无效的 # 字符可能由使用标记粘贴运算符 （#），而不是字符串化运算符的不正确宏插入 (）。
