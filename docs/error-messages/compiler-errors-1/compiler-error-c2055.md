# <a name="compiler-error-c2055"></a>编译器错误 C2055
预期的形式参数列表中，不是类型列表  
  
 函数定义包含参数类型列表而不是形参列表。 ANSI C 要求形参，除非它们是 void 或省略号命名为 (`...`)。  
  
 下面的示例生成 C2055:  
  
```  
// C2055.c  
// compile with: /c  
void func(int, char) {}  // C2055  
void func (int i, char c) {}   // OK  
```
