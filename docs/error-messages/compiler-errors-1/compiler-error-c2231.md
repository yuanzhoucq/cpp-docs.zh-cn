# <a name="compiler-error-c2231"></a>编译器错误 C2231
。︰ 左操作数指向 class-key，使用->  
  
 成员选择运算 （.） 左侧操作数是指针，而不是类、 结构或联合。  
  
 下面的示例生成 C2231:  
  
```  
// C2231.c  
struct S {  
   int member;  
} s, *ps = &s;  
int main() {  
   ps.member = 0;   // C2231  
  
   // OK  
   ps->member = 0;   // crash  
   s.member = 0;  
}  
```
