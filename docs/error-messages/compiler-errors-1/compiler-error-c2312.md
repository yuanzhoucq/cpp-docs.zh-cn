# <a name="compiler-error-c2312"></a>编译器错误 C2312
“exception1”: 由“exception2”在行号上捕获  
  
 两个处理程序捕获相同的异常类型。  
  
 下面的示例生成 C2312:  
  
```  
// C2312.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
    try {  
        throw "ooops!";  
    }  
    catch( signed int ) {}  
    catch( int ) {}   // C2312  
}  
```
