# <a name="compiler-error-c2293"></a>编译器错误 C2293
“identifie”：使成员变量作为 __based 说明符非法  
  
 说明符 `__based` 的修饰符必须为非成员指针。  
  
 以下示例生成 C2293：  
  
```  
// C2293.cpp  
// compile with: /c  
class A {  
   static int *i;  
   void __based(i) *bp;   // C2293  
   void *bp2;   // OK  
};  
```
