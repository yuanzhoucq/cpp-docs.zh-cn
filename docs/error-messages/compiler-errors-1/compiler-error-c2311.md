# <a name="compiler-error-c2311"></a>编译器错误 C2311

exception： 由...在行号上捕获

省略号 （...） 的 catch 处理程序必须以引发的最后一个处理程序。

下面的示例生成 C2311:

```
// C2311.cpp
// compile with: /EHsc
#include <eh.h>
int main() {
   try {
      throw "ooops!";
   }
   catch( ... ) {}
   catch( int ) {}   // C2311  ellipsis handler not last catch
}
```
