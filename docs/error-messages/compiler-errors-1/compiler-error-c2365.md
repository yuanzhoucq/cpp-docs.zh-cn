# <a name="compiler-error-c2365"></a>编译器错误 C2365
“class member”: 重定义；以前的定义是“class member”  
  
 你试图重新定义类成员。  
  
 下面的示例生成 C2365。  
  
```  
// C2365.cpp  
// compile with: /c  
class C1 {  
   int CFunc();  
   char *CFunc;   // C2365, already exists as a member function  
  
   int CMem;  
   char *CMem();   // C2365, already exists as a data member  
};  
```
