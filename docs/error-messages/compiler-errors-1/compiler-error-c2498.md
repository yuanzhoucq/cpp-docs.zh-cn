# <a name="compiler-error-c2498"></a>编译器错误 C2498
function: novtable 只能应用于类声明或定义  
  
 使用可导致此错误`__declspec(novtable)`使用函数。  
  
## <a name="example"></a>示例  
 下面的示例生成 C2498:  
  
```  
// C2498.cpp  
// compile with: /c  
void __declspec(novtable) f() {}   // C2498  
class __declspec(novtable) A {};   // OK  
```
