# <a name="compiler-error-c2194"></a>编译器错误 C2194

identifier： 是一个文本段

`data_seg`杂注使用与一起使用的段名称`code_seg`。

下面的示例生成 C2194:

```
// C2194.cpp
// compile with: /c
#pragma code_seg("MYCODE")
#pragma data_seg("MYCODE")   // C2194
#pragma data_seg("MYCODE2")   // OK
```
