# <a name="compiler-error-c2186"></a>编译器错误 C2186
“operator”：“void”类型的操作数非法  
  
 运算符有 `void` 操作数。  
  
 下面的示例生成 C2186：  
  
```  
// C2186.cpp  
// compile with: /c  
void func1( void );  
int  func2( void );  
int i = 2 + func1();   // C2186 func1() is type void  
int j = 2 + func2();   // OK both operands are type int  
```
