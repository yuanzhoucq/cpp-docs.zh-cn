# <a name="compiler-error-c2156"></a>编译器错误 C2156
杂注必须在函数的外部  
  
 必须在全局级别（在函数体外部）指定的杂注位于函数内。  
  
 下面的示例生成 C2156：  
  
```  
// C2156.cpp  
#pragma optimize( "l", on )   // OK  
int main() {  
   #pragma optimize( "l", on )   // C2156  
}  
```
