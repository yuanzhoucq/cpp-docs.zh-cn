# <a name="compiler-error-c2100"></a>编译器错误 C2100
非法的间接寻址  
  
 间接寻址运算符 ( `*` ) 应用于非指针值。  
  
 下面的示例生成 C2100:  
  
```  
// C2100.cpp  
int main() {  
   int r = 0, *s = 0;  
   s = &r;  
   *r = 200;   // C2100  
   *s = 200;   // OK  
}  
```
