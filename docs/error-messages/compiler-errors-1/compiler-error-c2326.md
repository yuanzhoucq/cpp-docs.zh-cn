# <a name="compiler-error-c2326"></a>编译器错误 C2326
“声明符”: 函数无法访问“名称”  
  
 该代码尝试修改成员变量，这是不可能的。  
  
## <a name="example"></a>示例  
 下面的示例生成 C2326:  
  
```  
// C2326.cpp  
void MyFunc() {  
   int i;  
  
   class MyClass  {  
   public:  
      void mf() {  
         i = 4;   // C2326 i is inaccessible  
      }  
   };  
}  
```
