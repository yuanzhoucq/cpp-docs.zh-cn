# 编译器错误 C2051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

case 表达式不是常数  
  
 Case 表达式必须是整数常数。  
  
 下面的示例生成 C2051：  
  
```  
// C2051.cpp  
class X {};  
  
int main() {  
   static X x;  
   int i = 0;  
  
   switch (i) {  
      case x:   // C2051 use constant expression to resolve error  
         break;  
      default:  
         break;  
   }  
}  
```  
  
 可能的解决方案：  
  
```  
// C2051b.cpp  
class X {};  
  
int main() {  
   static X x;  
   int i = 0;  
  
   switch (i) {  
      case 1:  
         break;  
      default:  
         break;  
   }  
}  
```
