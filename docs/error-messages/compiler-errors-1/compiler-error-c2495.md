# <a name="compiler-error-c2495"></a>编译器错误 C2495
identifier: nothrow 只能应用于函数声明或定义  
  
 [Nothrow](../../cpp/nothrow-cpp.md)扩展的特性可以应用于函数声明或定义仅。  
  
 下面的示例生成 C2495:  
  
```  
// C2495.cpp  
// compile with: /c  
__declspec(nothrow) class X {   // C2495  
   int m_data;  
} x;  
  
__declspec(nothrow) void test();   // OK  
```
