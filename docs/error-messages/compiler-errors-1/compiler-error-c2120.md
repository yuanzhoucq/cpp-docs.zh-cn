# <a name="compiler-error-c2120"></a>编译器错误 C2120
与所有类型的 void 非法  
  
 `void`具有另一种类型的声明中使用类型。  
  
 下面的示例生成 C2120:  
  
```  
// C2120.cpp  
int main() {  
   void int i;   // C2120  
   int j;   // OK  
}  
```
