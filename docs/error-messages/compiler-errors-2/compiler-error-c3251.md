# <a name="compiler-error-c3251"></a>编译器错误 C3251
无法调用在数值类型实例上的基类方法  
  
 由于 `GetClass` 是 `Microsoft.Runtime.Object`而不是 `Microsoft.Runtime.Integer4`的成员，所以将发生以下错误。
