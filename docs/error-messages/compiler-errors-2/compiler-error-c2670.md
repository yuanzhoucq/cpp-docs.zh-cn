# 编译器错误 C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“identifier”: 函数模板不能转换类型“type”的参数号  
  
 某参数未能转换为所需类型。  
  
 如果创建显式转换，可能会修复此错误。
