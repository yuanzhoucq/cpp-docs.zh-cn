# <a name="compiler-error-c2806"></a>编译器错误 C2806

operator operator 具有形参太多

重载的运算符具有太多参数。

下面的示例生成 C2806:

```
// C2806.cpp
// compile with: /c
class X {
public:
   X operator++ ( int, int );   // C2806 more than 1 parameter
   X operator++ ( int );   // OK
} ;
```
