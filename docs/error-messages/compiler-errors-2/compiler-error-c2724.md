# <a name="compiler-error-c2724"></a>编译器错误 C2724
identifier: 'static' 不应使用在文件范围内定义的成员函数  
  
 应使用外部链接声明静态成员函数。  
  
 下面的示例生成 C2724:  
  
```  
// C2724.cpp  
class C {  
   static void func();  
};  
  
static void C::func(){};   // C2724  
// try the following line instead  
// void C::func(){};  
```
