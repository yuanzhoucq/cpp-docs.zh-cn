# <a name="compiler-error-c2875"></a>编译器错误 C2875
使用声明将导致 class::identifier 的多个声明  
  
 声明将导致定义两次对同一个项。  
  
 下面的示例生成 C2875:  
  
```  
// C2875.cpp  
struct A {  
   void f(int*);  
};  
  
struct B {  
   void f(double*);  
};  
  
struct AB : A, B {  
   using A::f;  
   using A::f;   // C2875  
   using B::f;  
};  
```
