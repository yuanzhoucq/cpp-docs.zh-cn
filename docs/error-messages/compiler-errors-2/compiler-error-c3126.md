# <a name="compiler-error-c3126"></a>编译器错误 C3126

无法在托管类型 "type" 的内部定义联合 "union"

不能在托管类型内定义联合。

下面的示例生成 C3126：

```cpp
// C3126_2.cpp
// compile with: /clr /c
ref class Test
{
   union x
   {   // C3126
      int a;
      int b;
   };
};
```
