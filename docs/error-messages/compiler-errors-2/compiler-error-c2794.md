# <a name="compiler-error-c2794"></a>编译器错误 C2794
function： 不是 class 的任何直接或间接基类的成员  
  
 尝试使用[super](../../cpp/super.md)调用不存在的成员函数。  
  
 下面的示例生成 C2794  
  
```  
// C2794.cpp  
struct B {  
   void mf();  
};  
  
struct D : B {  
   void mf() {  
      __super::f();  // C2794  
   }  
};  
```
