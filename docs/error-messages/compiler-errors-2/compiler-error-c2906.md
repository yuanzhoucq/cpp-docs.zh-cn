# <a name="compiler-error-c2906"></a>编译器错误 C2906
“specialization”: 显式专用化需要“template <>”  
  
 有关模板的显式专用化，必须使用新的语法。  
  
 下面的示例生成 C2906:  
  
```  
// C2906.cpp  
// compile with: /c  
template<class T> class X{};   // primary template  
class X<int> { }   // C2906  
template<> class X<int> { };   // new syntax  
```
