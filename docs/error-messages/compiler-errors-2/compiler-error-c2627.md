# 编译器错误 C2627
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 匿名联合中不允许有成员函数  
  
 [anonymous union](../../misc/anonymous-unions.md) 不能具有成员函数。  
  
 下面的示例生成 C2627：  
  
```  
// C2627.cpp  
int main() {  
   union { void f(){} };   // C2627  
   union X { void f(){} };  
}  
```
