# <a name="compiler-error-c2640"></a>编译器错误 C2640
identifier: __based 修饰符非法引用  
  
 `__based`指针仅可使用修饰符。  
  
 下面的示例生成 C2640:  
  
```  
// C2640.cpp  
void f(int i) {  
    void *vp;  
    int _based(vp) &vr = I;  // C2640  
}  
```
