# <a name="compiler-error-c3265"></a>编译器错误 C3265
不能声明托管托管中构造的非托管的非托管构造'  
  
不能在非托管上下文中包含托管的对象。  
  
下面的示例重新产生 C3265:  
  
```  
// C3265_2.cpp  
// compile with: /clr /LD  
#include <vcclr.h>  
  
ref class A { };  
  
class B  
// try the following line instead  
// ref class B   
{  
   A ^a;   // C3265  
   // or embed the managed handle using gcroot  
   // try the following line instead  
   // gcroot<A^> a;  
};  
```
