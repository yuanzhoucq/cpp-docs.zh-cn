# <a name="compiler-error-c3001"></a>编译器错误 C3001
“error_text”: 应为 OpenMP 指令名称  
  
 `omp` 杂注后面必须跟有指令。  
  
 下面的示例生成 C3001：  
  
```  
// C3001.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp   // C3001 missing token  
}  
```
