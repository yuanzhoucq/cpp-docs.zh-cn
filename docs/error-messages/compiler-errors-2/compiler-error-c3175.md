# <a name="compiler-error-c3175"></a>编译器错误 C3175
function1︰ 不能将托管类型的方法调用从非托管函数 function2  
  
 非托管的函数不能调用托管类成员的函数。  
  
 下面的示例生成 C3175:  
  
```  
// C3175_2.cpp  
// compile with: /clr  
  
ref struct A {  
   static void func() {  
   }  
};  
  
#pragma unmanaged   // remove this line to resolve  
  
void func2() {  
   A::func();   // C3175  
}  
  
#pragma managed  
  
int main() {  
   A ^a = gcnew A;  
   func2();  
}  
```
