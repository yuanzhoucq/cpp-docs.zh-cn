# <a name="compiler-error-c2957"></a>编译器错误 C2957

“delim”: 无效的左侧分隔符: 应为“<”

泛型类格式不正确。

下面的示例生成 C2957：

```
// C2957.cpp
// compile with: /clr /LD
generic << class T>   // C2957
// try the following line instead
// generic < class T>
gc class C {};
```
