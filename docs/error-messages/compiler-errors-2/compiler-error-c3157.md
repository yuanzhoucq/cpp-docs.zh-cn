# <a name="compiler-error-c3157"></a>编译器错误 C3157

ParamArray 特性只能应用到的最后一个参数

[ParamArray](https://msdn.microsoft.com/library/system.paramarrayattribute.aspx)属性被应用到了错误的参数。
