# <a name="compiler-error-c3007"></a>编译器错误 C3007
“arg”: OpenMP“directive”指令上的子句没有采用参数  
  
 OpenMP 指令具有一个参数，但没有采用参数。  
  
 以下示例生成 C3007：  
  
```  
// C3007.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp parallel for ordered(2)   // C3007  
}  
```
