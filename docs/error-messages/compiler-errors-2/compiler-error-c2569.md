# <a name="compiler-error-c2569"></a>编译器错误 C2569
EnumOrUnion： 枚举/联合不能用作基类  
  
 如果你必须派生自指定的联合或枚举类型，将更改该联合或枚举，为类或结构。  
  
 下面的示例生成 C2569:  
  
```  
// C2569.cpp  
// compile with: /c  
union ubase {};  
class cHasPubUBase : public ubase {};   // C2569  
// OK  
struct sbase {};  
class cHasPubUBase : public sbase {};  
```
