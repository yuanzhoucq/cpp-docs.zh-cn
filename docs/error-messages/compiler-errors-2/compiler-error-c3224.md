# <a name="compiler-error-c3224"></a>编译器错误 C3224
“type”：没有任何重载泛型类采用“number”泛型类型参数  
  
 编译器找不到适当的重载。  
  
 下面的示例生成 C3224：  
  
```  
// C3224.cs  
// compile with: /target:library  
public class C<T> {}  
public class C<T,U> {}  
```  
  
 然后，  
  
```  
// C3224b.cpp  
// compile with: /clr  
#using "C3224.dll"  
int main() {  
   C<int,int,int>^ c = gcnew C<int,int,int>();   // C3224  
   C<int,int>^ c2 = gcnew C<int,int>();   // OK  
}  
```
