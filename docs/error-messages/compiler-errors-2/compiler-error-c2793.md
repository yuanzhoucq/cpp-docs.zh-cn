# <a name="compiler-error-c2793"></a>编译器错误 C2793
token： 意外的令牌以下::，标识符或关键字 operator 预期  
  
 可以遵循的标记仅`__super::`是标识符或关键字`operator`。  
  
 下面的示例生成 C2793  
  
```  
// C2793.cpp  
struct B {  
   void mf();  
};  
  
struct D : B {  
   void mf() {  
      __super::(); // C2793  
   }  
};  
```
