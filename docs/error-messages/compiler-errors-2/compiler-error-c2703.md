# <a name="compiler-error-c2703"></a>编译器错误 C2703

非法的 __leave 语句

一个`__leave`语句必须位于`__try`块。

下面的示例生成 C2703:

```
// C2703.cpp
int main() {
   __leave;   // C2703
   __try {
      // try the following line instead
      __leave;
   }
   __finally {}
}
```
