# <a name="compiler-error-c2655"></a>编译器错误 C2655

identifier： 定义或在当前作用域中的非法重新声明

可以仅在全局范围内重新声明标识符。

下面的示例生成 C2655:

```
// C2655.cpp
class A {};
class B {
public:
   static int i;
};

int B::i;  // OK

int main() {
   A B::i;  // C2655
}
```
