# <a name="compiler-error-c2792"></a>编译器错误 C2792

super： 此关键字后面必须跟::

可以按照关键字的唯一令牌`__super`是`::`。

下面的示例生成 C2792:

```
// C2792.cpp
struct B {
   void mf();
};

struct D : B {
   void mf() {
      __super.();   // C2792

      // try the following line instead
      // __super::mf();
   }
};
```
