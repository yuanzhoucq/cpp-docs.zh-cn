# <a name="compiler-error-c3239"></a>编译器错误 C3239

“type”：公共语言运行时不允许使用指向内部/pin 指针的指针

编译器遇到了无效的类型。

下面的示例生成 C3229：

```
// C3239.cpp
// compile with: /clr
int main() {
   interior_ptr<int>* pip0;   // C3239

   // OK
   int * pip1;
   interior_ptr<int> pip2;
   int ** pip;
}
```
