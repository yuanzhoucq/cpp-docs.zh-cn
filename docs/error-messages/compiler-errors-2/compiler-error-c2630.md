# 编译器错误 C2630
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

在应是逗号分隔列表的位置找到“symbol”  
  
 该符号出现在需要逗号的上下文中。  
  
 下面的示例生成 C2630：  
  
```  
// C2630.cpp  
// compile with: /c  
struct D {  
   D(int);  
};  
  
struct E {  
   E(int);  
};  
  
class C : public D, public E {  
   C();  
};  
  
C::C() : D(0) ; E(0) { }   // C2630  
C::C() : D(0), E(0) {}   // OK  
```
