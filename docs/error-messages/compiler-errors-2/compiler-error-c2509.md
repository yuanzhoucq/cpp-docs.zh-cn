# <a name="compiler-error-c2509"></a>编译器错误 C2509

identifier： 未在 class 中声明的成员函数

未在指定的类中声明该函数。

## <a name="example"></a>示例

下面的示例生成 C2509。

```
// C2509.cpp
// compile with: /c
struct A {
   virtual int vfunc() = 0;
   virtual int vfunc2() = 0;
};

struct B : private A {
   using A::vfunc;
   virtual int vfunc2();
};

int B::vfunc() { return 1; }   // C2509
int B::vfunc2() { return 1; }   // OK
```
