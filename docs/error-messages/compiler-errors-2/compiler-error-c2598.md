# <a name="compiler-error-c2598"></a>编译器错误 C2598

链接规范必须在全局范围内

链接说明符声明在本地作用域内。

下面的示例生成 C2598:

```
// C2598.cpp
// compile with: /c
void func() {
   extern "C" int func2();   // C2598
}

extern "C" int func( int i );
```
