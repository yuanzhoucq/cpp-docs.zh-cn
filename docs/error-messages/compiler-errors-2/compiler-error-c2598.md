# 编译器错误 C2598
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

链接规范必须在全局范围内  
  
 在局部范围中声明了链接说明符。  
  
 下面的示例生成 C2598：  
  
```  
// C2598.cpp  
// compile with: /c  
void func() {  
   extern "C" int func2();   // C2598  
}  
  
extern "C" int func( int i );  
```
