# 编译器错误 C2934
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“class”：type\-class\-id 被重新定义为“identifier”的嵌套“item”  
  
 不能将泛型或模板类用作嵌套项。
