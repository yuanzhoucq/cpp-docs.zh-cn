# 编译器错误 C3769
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“type”: 嵌套类不能与直接封闭类同名  
  
 嵌套类不能与直接封闭类同名。  
  
## 示例  
 下面的示例生成 C3769。  
  
```  
// C3769.cpp  
// compile with: /c  
class x {  
   class x {};   // C3769  
   class y {  
      class x{};   // OK  
   };  
};  
```
