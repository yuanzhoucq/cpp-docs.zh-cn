# <a name="compiler-error-c3740"></a>编译器错误 C3740

模板不能发出或接收事件

模板化类或结构不能包含[事件](../../cpp/event-handling.md)。

下面的示例生成 C3740:

```
// C3740.cpp
template <typename T>   // Delete the template specification
struct E {
   __event void f();   // C3740
};

int main() {
}
```
