# <a name="compiler-error-c2579"></a>编译器错误 C2579

无法解析类型 type （偏移量）。 应在文件中

C2579 始终遵循 C4691。 有关详细信息，请参阅[编译器警告 （等级 1） C4691](../../error-messages/compiler-warnings/compiler-warning-level-1-c4691.md)。
