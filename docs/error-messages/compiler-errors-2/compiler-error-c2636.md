# <a name="compiler-error-c2636"></a>编译器错误 C2636

identifier： 指向引用成员的指针是非法的

已声明为指向引用成员的指针。

下面的示例生成 C2636:

```
// C2636.cpp
struct S {};
int main() {
   int &S::*prs;   // C2636
   int S::*prs1;   // OK
   int *S::*prs2;   // OK
}
```
