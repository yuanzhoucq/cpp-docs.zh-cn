# <a name="compiler-error-c3223"></a>编译器错误 C3223
“property”：不能将“typeid”应用于属性  
  
 不能将 [typeid](../../windows/typeid-cpp-component-extensions.md) 应用于属性。  
  
## <a name="example"></a>示例  
 以下示例生成 C3223。  
  
```  
// C3223.cpp  
// compile with: /clr  
ref class R {  
public:  
   property int myprop;  
};  
  
int main() {  
   System::Type^ type2 = R::myprop::typeid;   // C3223  
}  
```
