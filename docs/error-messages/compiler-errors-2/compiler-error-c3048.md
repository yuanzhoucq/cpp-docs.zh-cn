# <a name="compiler-error-c3048"></a>编译器错误 C3048

“#pragma omp atomic”后面的表达式格式不正确

未正确指定了原子指令。

下面的示例生成 C3048：

```
// C3048.cpp
// compile with: /openmp vcomps.lib
#include "omp.h"
#include <stdio.h>

int main() {
   int a[10];
   omp_set_num_threads(4);
   #pragma omp parallel
   {
      #pragma omp atomic
      a[0] = 1;   // C3048
      // try the following line instead
      // a[0] += 1;
   }
}
```
