# <a name="compiler-error-c3834"></a>编译器错误 C3834
显式转换为钉住指针非法；改用钉住的局部变量  
  
 不允许显式强制转换为固定指针。  
  
## <a name="example"></a>示例  
 下面的示例生成 C3834。  
  
```  
// C3834.cpp  
// compile with: /clr  
int main() {  
   int x = 33;  
  
   pin_ptr<int> p = safe_cast<pin_ptr<int> >(&x);   // C3834  
   pin_ptr<int> p2 = &x;   // OK  
}  
```
