# <a name="compiler-error-c2830"></a>编译器错误 C2830
只有“operator new”的位置参数可以有默认值  
  
标准形参[运算符 new](../../standard-library/new-operators.md#operator_new)不能有默认值。 只有用户定义的位置参数可以指定默认值。
