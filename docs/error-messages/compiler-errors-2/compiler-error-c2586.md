# <a name="compiler-error-c2586"></a>编译器错误 C2586
不正确的用户定义的转换语法： 非法的间接寻址  
  
 不允许的转换运算符的间接寻址。  
  
 下面的示例生成 C2586:  
  
```  
// c2586.cpp  
// compile with: /c  
struct C {  
   * operator int();   // C2586  
   operator char();   // OK  
};  
```
