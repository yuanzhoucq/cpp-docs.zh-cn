# 编译器错误 C3139
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“struct”: 无法导出没有成员的 UDT  
  
 尝试对空 UDT（用户定义的类型）应用 [export](../../windows/export.md) 特性。  例如：  
  
```  
// C3139.cpp  
#include "unknwn.h"  
[emitidl];  
[module(name=xx)];  
  
[export] struct MyStruct {   // C3139 empty type  
};  
int main(){}  
```
