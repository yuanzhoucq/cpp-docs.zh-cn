# <a name="compiler-error-c2731"></a>编译器错误 C2731
identifier： 无法重载函数  
  
 函数`main`， `WinMain`， `DllMain`，和`LibMain`无法进行重载。  
  
 下面的示例生成 C2731:  
  
```  
// C2731.cpp  
extern "C" void WinMain(int, char *, char *);  
void WinMain(int, short, char *, char*);   // C2731  
```
