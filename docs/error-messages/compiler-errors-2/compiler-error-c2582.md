# <a name="compiler-error-c2582"></a>编译器错误 C2582

function 函数类型中不可用

尝试将分配到不具有赋值运算符的对象。

下面的示例生成 C2582:

```
// C2582.cpp
// compile with: /clr
using namespace System;

struct N {};
ref struct O {};
ref struct R {
   property O prop;   // C2582
   property O ^ prop2;   // OK
};

int main() {
   String ^ st1 = gcnew String("");
   ^st1 = gcnew String("");   // C2582
   st1 = "xxx";   // OK
}
```
