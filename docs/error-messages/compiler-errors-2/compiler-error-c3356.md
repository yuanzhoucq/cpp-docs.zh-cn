# 编译器错误 C3356
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“attribute”: 无法使用完全限定名调用多路广播特性  
  
 未正确指定由多个进程（例如编译器和 ATL 提供程序）处理的特性。
