# <a name="compiler-error-c3005"></a>编译器错误 C3005
“error_text”：OpenMP“directive”指令上出现意外的标记  
  
 OpenMP 指令格式不正确。  
  
 下面的示例生成 C3005：  
  
```  
// C3005.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp parallel + for   // C3005  
}  
```  
  
 如果将左大括号放置在杂注所在的行，也会发生 C3005。  
  
```  
// C3005b.c  
// compile with: /openmp  
int main() {  
   #pragma omp parallel {   // C3005 put open brace on next line  
   lbl2:;  
   }  
   goto lbl2;  
}  
```
