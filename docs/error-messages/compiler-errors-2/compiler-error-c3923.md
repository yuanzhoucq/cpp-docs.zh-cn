# <a name="compiler-error-c3923"></a>编译器错误 C3923

“member”：不允许在 WinRT 或托管类的成员函数中使用局部类、结构或联合定义

## <a name="example"></a>示例

下列示例生成 C3923。

```cpp
// C3923.cpp
// compile with: /clr /c
ref struct x {
   void Test() {
      struct a {};   // C3923
      class b {};   // C3923
      union c {};   // C3923
   }
};
```
