# <a name="compiler-error-c3275"></a>编译器错误 C3275
“enum member”：无法使用该无限定符的符号  
  
使用托管代码以及两个或多个枚举包含具有相同名称的标识符时，必须显式限定对标识符的引用。  
  
C3275 才可访问使用过时的编译器选项**/clr:oldSyntax**。
