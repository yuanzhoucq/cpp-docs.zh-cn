# <a name="compiler-error-c3113"></a>编译器错误 C3113
结构不能为模板/泛型  
  
 你尝试生成类模板或类泛型接口或枚举外。  
  
 下面的示例生成 C3113:  
  
```  
// C3113.cpp  
// compile with: /c  
template <class T>   
enum E {};   // C3113  
// try the following line instead  
// class MyClass{};  
```
