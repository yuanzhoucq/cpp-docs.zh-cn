# <a name="compiler-error-c3015"></a>编译器错误 C3015
OpenMP“for”语句中的初始化格式不正确  
  
 必须完全或显式指定 OpenMP 语句中的 `for` 循环。  
  
 以下示例生成 C3015：  
  
```  
// C3015.cpp  
// compile with: /openmp  
int main()  
{  
   int i = 0, j = 10;  
  
   #pragma omp parallel  
   {  
      #pragma omp for  
      for (; i < 0; i += j)   // C3015  
      // Try the following line instead:  
      // for (i = 0; i < 0; i++)   
         --j;  
   }  
}  
```
