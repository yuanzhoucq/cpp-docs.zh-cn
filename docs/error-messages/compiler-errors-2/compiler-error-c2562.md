# <a name="compiler-error-c2562"></a>编译器错误 C2562

identifier: void 函数返回值

该函数声明为`void`但返回一个值。

此错误可能引起错误的函数原型。

如果在函数声明中指定的返回类型，可能会修复此错误。

下面的示例生成 C2562:

```
// C2562.cpp
// compile with: /c
void testfunc() {
   int i;
   return i;   // C2562 delete the return to resolve
}
```
