# <a name="compiler-error-c2654"></a>编译器错误 C2654

identifier： 成员函数的外部访问成员的尝试

在声明中访问成员。 仅在成员函数中，可以访问成员数据。

尝试在声明中初始化变量时，可以导致此错误。 使用构造函数实现此目的。
