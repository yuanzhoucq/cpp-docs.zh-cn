# <a name="compiler-error-c3045"></a>编译器错误 C3045
OpenMP “sections”指令后应为一个复合语句。 缺少“{”  
  
 大括号分隔的代码块必须跟[部分](../../parallel/openmp/reference/sections-openmp.md)指令。  
  
 下面的示例生成 C3045：  
  
```  
// C3045.cpp  
// compile with: /openmp /c  
#include "omp.h"  
  
int main() {  
   int n2 = 2, n3 = 3;  
  
   #pragma omp parallel  
   {  
      ++n2;  
  
      #pragma omp sections  
         ++n2;   // C3045  
  
      #pragma omp sections   // OK  
      {  
         ++n3;  
      }  
   }  
}  
```
