# <a name="compiler-error-c3414"></a>编译器错误 C3414
member： 不能定义导入的成员函数  
  
 此外定义中引用的程序集中的代码中定义的成员。  
  
 下面的示例生成 C3414:  
  
```  
// C3414a2.cpp  
// compile with: /clr /LD  
public ref class MyClass {  
public:  
   void Test(){}  
};  
```  
  
 然后：  
  
```  
// C3414b2.cpp  
// compile with: /clr  
#using <C3414a2.dll>  
  
void MyClass::Test() {    // C3414  
}  
  
System::Object::Object() {    // C3414  
}  
```
