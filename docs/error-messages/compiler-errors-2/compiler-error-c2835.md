# <a name="compiler-error-c2835"></a>编译器错误 C2835

用户定义的转换 type 不接受形参

用户定义的类型转换不能采用正式的参数。

下面的示例生成 C2835:

```
// C2835.cpp
class A {
public:
   char v_char;

   A() {
      v_char = 'A';
   };
   operator char(char a) {   // C2835
   // try the following line instead
   // operator char() {
      return v_char + 1;
   };
};

int main() {
   A a;
}
```
