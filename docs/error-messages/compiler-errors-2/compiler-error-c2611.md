# 编译器错误 C2611
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“token”: 非法的后缀“~”\(应输入标识符\)  
  
 该标记不是标识符。  
  
 下面的示例生成 C2611：  
  
```  
// C2611.cpp  
// compile with: /c  
class C {  
   C::~operator int();   // C2611  
   ~C();   // OK  
};  
```
