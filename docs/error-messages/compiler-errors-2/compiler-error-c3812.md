# <a name="compiler-error-c3812"></a>编译器错误 C3812
必须在属性声明中的第一个标记属性。  
  
 当声明的属性，`__property`关键字必须是在行上的第一个标记。  
  
 C3812 才可使用已过时的编译器选项连接**/clr:oldSyntax**。
