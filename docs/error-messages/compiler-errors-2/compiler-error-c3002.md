# <a name="compiler-error-c3002"></a>编译器错误 C3002
“name1 name2”：多个 OpenMP 指令名称  
  
 不允许使用多个指令名称。  
  
 下面的示例生成 C3002：  
  
```  
// C3002.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp parallel single   // C3002  
}  
```
