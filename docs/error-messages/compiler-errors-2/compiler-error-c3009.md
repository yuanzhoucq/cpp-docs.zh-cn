# <a name="compiler-error-c3009"></a>编译器错误 C3009
“label”: 不允许跳转到 OpenMP 结构化块中  
  
 代码不能跳转到或跳出 OpenMP 块。  
  
 以下示例生成 C3009：  
  
```  
// C3009.c  
// compile with: /openmp  
int main() {  
   #pragma omp parallel   
   {  
   lbl2:;  
   }  
   goto lbl2;   // C3009  
}  
```
