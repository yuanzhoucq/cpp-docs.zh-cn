# <a name="compiler-error-c2751"></a>编译器错误 C2751

parameter： 无法限定函数参数的名称

限定的名称不能用作函数参数。

下面的示例生成 C2751:

```
// C2751.cpp
namespace std {
   template<typename T>
   class list {};
}

#define list std::list
void f(int &list){}   // C2751
```
