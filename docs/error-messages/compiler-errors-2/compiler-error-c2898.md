# <a name="compiler-error-c2898"></a>编译器错误 C2898
declaration： 不能是虚拟成员函数模板  
  
 下面的示例生成 C2898:  
  
```  
// C2898.cpp  
// compile with: /c  
class X {  
public:  
   template<typename T> virtual void f(T t) {}   // C2898  
};  
```
