# <a name="compiler-error-c3214"></a>编译器错误 C3214
“type”：泛型“generic_type”的泛型形参“param”的类型实参无效，不符合“constraint”约束  
  
 对不符合泛型类的约束的泛型类的实例化指定该类型。  
  
 以下示例生成 C3214：  
  
```  
// C3214.cpp  
// compile with: /clr  
interface struct A {};  
  
generic <class T>   
where T : A  
ref class C {};  
  
ref class X : public A {};  
  
int main() {  
   C<int>^ c = new C<int>;   // C3214  
   C<X ^> ^ c2 = new C<X^>;   // OK  
}  
```
