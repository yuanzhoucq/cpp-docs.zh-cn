# <a name="compiler-error-c3273"></a>编译器错误 C3273
__finally 不能用于非托管代码中的异常块。  
  
 下面的示例生成 C3273：  
  
```  
// C3273.cpp  
// compile with: /GX  
int main()  
{     
   try  
   {  
   }  
   catch (int)  
   {  
   }  
   __finally   // C3273, remove __finally clause  
   {  
   }  
}  
```
