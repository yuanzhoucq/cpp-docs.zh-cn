# <a name="compiler-error-c3360"></a>编译器错误 C3360

“字符串”：无法创建名称

传递给 [uuid](../../windows/uuid-cpp-attributes.md) 特性的值无效。

以下示例生成 C3360：

```
// C3360.cpp
[ uuid("1") ]
// try this line instead
// [ uuid("12341234-1234-1234-1234-123412341234") ]
struct A   // C3360
{
};

int main()
{
}
```
