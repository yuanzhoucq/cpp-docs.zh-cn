# <a name="compiler-error-c2702"></a>编译器错误 C2702
__except 中不能出现终止块  
  
 异常处理程序 (`__try`/`__except`) 不能嵌套在`__finally`块。  
  
 下面的示例生成 C2702:  
  
```  
// C2702.cpp  
// processor: x86 IPF  
int Counter;  
int main() {  
   __try {}  
   __finally {  
      __try {}   // C2702  
      __except( Counter ) {}   // C2702  
   }  
}  
```
