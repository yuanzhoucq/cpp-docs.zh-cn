# 编译器错误 C2721
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“specifier”: 运算符关键字和类型之间的存储类说明符非法  
  
 用户定义的类型转换适用于所有存储类，所以无法在类型转换中指定存储类。
