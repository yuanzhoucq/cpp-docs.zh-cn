# <a name="compiler-error-c3010"></a>编译器错误 C3010
“label”：不允许跳出 OpenMP 结构化块  
  
 代码不能跳转到或跳出 OpenMP 块。  
  
 以下示例生成 C3010：  
  
```  
// C3010.c  
// compile with: /openmp  
int main() {  
   #pragma omp parallel   
   {  
      #pragma omp parallel  
      {  
         goto lbl3;  
      }  
   }  
   lbl3:;   // C3010  
}  
```
