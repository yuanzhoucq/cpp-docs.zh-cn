# <a name="compiler-error-c2689"></a>编译器错误 C2689

function： 不能局部类中定义友元函数

您可以声明，但不是在本地类中定义友元函数。

下面的示例生成 C2689:

```
// C2689.cpp
// compile with: /c
void g() {
   void f2();
   class X {
      friend void f2(){}   // C2689
      friend void f2();   // OK
   };
}
```
