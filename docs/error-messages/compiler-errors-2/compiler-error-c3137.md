# <a name="compiler-error-c3137"></a>编译器错误 C3137

property： 无法初始化属性

无法初始化属性，例如，在构造函数的初始化列表中。

下面的示例生成 C3137:

```
// C3137.cpp
// compile with: /clr /c
ref class CMyClass {
public:
   property int Size {
      int get() {
         return 0;
      }
      void set( int i ) {}
   }

   CMyClass() : Size( 1 ) {   // C3137
      // to resolve this C3137, remove the initializer from the
      // ctor declaration and perform the assignment as follows
      // Size = 1;
   }
};
```
