# <a name="compiler-error-c3008"></a>编译器错误 C3008

“arg”: OpenMP“directive”指令上的参数缺少右括号“)”

获取参数的 OpenMP 指令缺少右括号。

以下示例生成 C3008:

```
// C3008.c
// compile with: /openmp
int main()
{
   int x, y, z;
   #pragma omp parallel shared(x   // C3008
   // Try the following line instead:
   #pragma omp parallel shared(x)
   {
   }
}
```
