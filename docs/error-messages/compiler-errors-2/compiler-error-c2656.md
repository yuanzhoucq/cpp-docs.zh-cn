# 编译器错误 C2656
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“function”: 函数不能作为位域使用  
  
 函数被声明为位域的成员。  
  
 此错误可能由构造函数初始值设定项列表中的语法错误引起。
