# <a name="compiler-error-c3211"></a>编译器错误 C3211
“显式专用化”: 显式专用化正在使用部分专用化语法，请改用模板 <>  
  
 显式专用化的格式不正确。  
  
 以下示例生成 C3211：  
  
```  
// C3211.cpp  
// compile with: /LD  
template<class T>  
struct s;  
  
template<class T>  
// use the following line instead  
// template<>  
struct s<int>{};   // C3211  
```
