# <a name="compiler-error-c2867"></a>编译器错误 C2867
identifier： 不是命名空间  
  
 A`using`指令应用于命名空间之外的内容。  
  
 下面的示例生成 C2867:  
  
```  
// C2867.cpp  
// compile with: /c  
namespace N {  
   class X {};  
}  
using namespace N::X;   // C2867  
```
