# <a name="compiler-error-c3135"></a>编译器错误 C3135
property: 属性不能具有 const 或易失性键入  
  
 [Const](../../cpp/const-cpp.md)和[易失性](../../cpp/volatile-cpp.md)属性上不允许关键字。
