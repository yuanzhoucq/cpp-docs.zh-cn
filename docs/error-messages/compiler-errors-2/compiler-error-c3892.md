# <a name="compiler-error-c3892"></a>编译器错误 C3892
var： 不能将其分配给为常量类型的变量  
  
 声明和初始化后，不能更改 const 变量。  
  
 下面的示例生成 C3892:  
  
```  
// C3892.cpp  
// compile with: /clr  
ref struct Y1 {  
   static const int staticConst = 9;  
};  
  
int main() {  
   Y1::staticConst = 0;   // C3892  
}  
```
