# <a name="compiler-error-c3192"></a>编译器错误 C3192

语法错误： "^" 不是前缀运算符（您是否希望使用 "*"？）

句柄不能用作取消引用运算符。

下面的示例生成 C3192：

```cpp
// C3192.cpp
// compile with: /clr
using namespace System;

ref class MyClass {
public:
   MyClass () {}
   MyClass(MyClass%) {}
};

int main() {
   MyClass ^ s = gcnew MyClass;
   MyClass b = ^s;   // C3192

   // OK
   MyClass b2 = *s;
}
```
