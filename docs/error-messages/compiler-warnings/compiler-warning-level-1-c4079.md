# 编译器警告（等级 1）C4079
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

意外的标记“token”  
  
 杂注参数列表中出现意外的分隔符标记。  忽略杂注的其余部分。  
  
 下面的示例生成 C4079：  
  
```  
// C4079.cpp  
// compile with: /W1  
#pragma warning(disable : 4081)  
#pragma pack(c,16)   // C4079  
  
int main() {  
}  
```
