# <a name="compiler-warning-level-4-c4725"></a>编译器警告（等级 4）C4725
在一些 Pentium 中，指令可能不准确  
  
 你的代码包含某种内联程序集指令，其可能在某些 Pentium 微处理器上不会产生准确结果。  
  
 以下示例生成 C4725：  
  
```  
// C4725.cpp  
// compile with: /W4  
// processor: x86  
double m32fp = 2.0003e-17;  
  
void f() {  
   __asm  
   {  
      FDIV m32fp   // C4725  
   }  
}  
  
int main() {  
}  
```
