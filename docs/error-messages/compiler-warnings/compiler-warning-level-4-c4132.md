# 编译器警告（等级 4）C4132
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“object”：应初始化 const 对象  
  
 未初始化常量。 在对具有“const”属性的符号进行定义后，不能更改它的值，应向它提供一个初始化值。
