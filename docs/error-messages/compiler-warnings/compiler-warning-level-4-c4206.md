# <a name="compiler-warning-level-4-c4206"></a>编译器警告（等级 4）C4206

**使用了非标准扩展：翻译单元为空**

预处理后，该文件为空。

此扩展会阻止你的代码移植到其他编译器。 它在 ANSI 兼容性（[/za](../../build/reference/za-ze-disable-language-extensions.md)）下生成错误，并且仅适用于 C 源代码。
