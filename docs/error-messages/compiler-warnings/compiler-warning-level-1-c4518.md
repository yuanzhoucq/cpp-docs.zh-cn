# <a name="compiler-warning-level-1-c4518"></a>编译器警告（等级 1）C4518
specifier： 存储类或意外的说明符在此处键入。忽略  
  
 下面的示例生成 C4518:  
  
```  
// C4518.cpp  
// compile with: /c /W1  
_declspec(dllexport) extern "C" void MyFunction();   // C4518  
  
extern "C" void MyFunction();   // OK  
```
