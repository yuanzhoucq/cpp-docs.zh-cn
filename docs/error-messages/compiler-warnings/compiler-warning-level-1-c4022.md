# <a name="compiler-warning-level-1-c4022"></a>编译器警告 （等级 1） C4022
function： 指针的实际参数 number 的不匹配  
  
 实际的参数的指针类型不同于相应的形式参数的指针类型。 实际的参数传递时未经更改。
