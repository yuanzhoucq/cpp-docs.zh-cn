# <a name="compiler-warning-level-3-c4133"></a>编译器警告（等级3） C4133

"type"：类型不兼容-从 "type1" 到 "type2"

尝试减去两个不同类型的指针会导致此警告。

若要避免出现此警告，请提供适当的类型转换。
