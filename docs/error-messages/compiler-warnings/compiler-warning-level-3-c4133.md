# <a name="compiler-warning-level-3-c4133"></a>编译器警告 （等级 3） C4133

type： 不兼容的类型-从 type1 到 type2

可以通过尝试不同类型的两个指针进行减法运算导致该警告。

若要避免此警告，请提供适当的类型转换。
