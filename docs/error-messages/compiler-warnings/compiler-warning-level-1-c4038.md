# <a name="compiler-warning-level-1-c4038"></a>编译器警告（等级 1）C4038
“modifier”：非法的环境类修饰符  
  
 此修饰符不能使用为类**dllimport**或[dllexport](../../cpp/dllexport-dllimport.md)属性。
