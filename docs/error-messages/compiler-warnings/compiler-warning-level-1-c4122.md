# <a name="compiler-warning-level-1-c4122"></a>编译器警告（等级 1）C4122
“function”: alloc_text 仅适用于带 C 链接的函数  
  
 **alloc_text** 杂注仅适用于使用 **extern c**声明的函数。 它不能与外部 C++ 函数一起使用。  
  
 将忽略杂注。
