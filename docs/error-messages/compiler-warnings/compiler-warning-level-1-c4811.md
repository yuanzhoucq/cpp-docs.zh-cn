# <a name="compiler-warning-level-1-c4811"></a>编译器警告（等级 1）C4811

杂注 conform(forScope, show) 的值 == value

你使用 **conform** 杂注的 [“显示”](../../preprocessor/conform.md) 选项，将发出此警告。 *value* 为当前的 conform 值。
