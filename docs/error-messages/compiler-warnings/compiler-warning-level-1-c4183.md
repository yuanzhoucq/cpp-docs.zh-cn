# <a name="compiler-warning-level-1-c4183"></a>编译器警告 （等级 1） C4183
identifier： 缺少返回的类型;假定是返回 int 的成员函数  
  
 类或结构中的成员函数的内联定义没有返回类型。 假定该成员函数返回类型的默认`int`。  
  
 下面的示例生成 C4183:  
  
```  
// C4183.cpp  
// compile with: /W1 /c  
#pragma warning(disable : 4430)  
class MyClass1;  
class MyClass2 {  
   MyClass1() {};   // C4183  
};  
```
