# <a name="compiler-warning-level-4-c4505"></a>编译器警告（等级 4）C4505

function： 未引用本地函数已移除

给定的函数是模块的本地且不被引用中; 在正文中因此，该函数是死代码。

编译器未生成此死函数代码。
