# <a name="compiler-warning-level-1-c4402"></a>编译器警告（等级 1）C4402

必须使用 PTR 运算符

在内联程序集代码中引用或强制转换为类型时，不带 PTR 运算符的操作数上使用了类型。

编译器假定为 PTR 运算符。
