# <a name="compiler-warning-level-1-c4402"></a>编译器警告（等级 1）C4402

必须使用 PTR 运算符

而无需引用或内联程序集代码中的类型强制转换时的 PTR 运算符的操作数上使用的类型。

编译器将假定 PTR 运算符。
