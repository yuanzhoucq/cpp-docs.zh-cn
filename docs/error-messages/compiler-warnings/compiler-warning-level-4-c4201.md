# <a name="compiler-warning-level-4-c4201"></a>编译器警告（等级 4）C4201
使用的非标准扩展： 无名称结构/联合  
  
 在 Microsoft 扩展 (/Ze) 中，可以指定没有声明符的结构，作为另一个结构或联合的成员。 这些结构生成在 ANSI 兼容性错误 ([/Za](../../build/reference/za-ze-disable-language-extensions.md))。  
  
## <a name="example"></a>示例  
  
```  
// C4201.cpp  
// compile with: /W4  
struct S  
{  
   float y;  
   struct  
   {  
      int a, b, c;  // C4201  
   };  
} *p_s;  
  
int main()  
{  
}  
```
