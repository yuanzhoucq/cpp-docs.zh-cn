# <a name="compiler-warning-level-1-c4187"></a>编译器警告（等级 1）C4187
\#导入特性 attribute1 和 attribute2 不兼容;同时忽略  
  
 A [#import](../../preprocessor/hash-import-directive-cpp.md)指定语句`no_implementation`和`implementation_only`属性。 忽略这两个特性。
