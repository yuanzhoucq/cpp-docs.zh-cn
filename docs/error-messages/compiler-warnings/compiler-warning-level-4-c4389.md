# <a name="compiler-warning-level-4-c4389"></a>编译器警告（等级 4）C4389

operator： 有符号/无符号不匹配

操作涉及符号和无符号的变量。 这可能导致数据丢失。

下面的示例生成 C4389:

```
// C4389.cpp
// compile with: /W4
#pragma warning(default: 4389)

int main()
{
   int a = 9;
   unsigned int b = 10;
   if (a == b)   // C4389
      return 0;
   else
      return 0;
};
```
