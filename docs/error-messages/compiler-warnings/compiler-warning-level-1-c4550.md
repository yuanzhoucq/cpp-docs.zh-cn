# <a name="compiler-warning-level-1-c4550"></a>编译器警告（等级 1）C4550

表达式计算为缺少参数列表的函数

取消引用的指向函数为缺少参数列表。

## <a name="example"></a>示例

```
// C4550.cpp
// compile with: /W1
bool f()
{
   return true;
}

typedef bool (*pf_t)();

int main()
{
   pf_t pf = f;
   if (*pf) {}  // C4550
   return 0;
}
```
