# <a name="compiler-warning-level-1-c4179"></a>编译器警告（等级 1）C4179

> "//*"：被分析为 "/" 和 "/\*"：与标准 "//" 注释混淆

__//\*__ 是不正确的注释分隔符。 请改用 __//__ 或 __/\*__ 。
