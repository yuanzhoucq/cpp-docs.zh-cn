# 编译器警告（等级 1）C4119
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

指定了不同的基“base1”和“base2”  
  
 两个基指针不兼容，因为它们具有不同的基。 编译器不能在它们之间转换。
