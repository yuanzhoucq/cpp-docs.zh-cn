# <a name="compiler-warning-level-1-c4168"></a>编译器警告（等级 1）C4168
编译器限制：超出调试器类型，删除程序数据库“database”并重新生成  
  
 必须重新生成程序数据库文件，以在程序中容纳所有类型。
