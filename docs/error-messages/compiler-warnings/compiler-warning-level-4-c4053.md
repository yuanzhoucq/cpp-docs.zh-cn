# 编译器警告（等级 4）C4053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“?:”的一个 void 操作数  
  
 为 `?:` 运算符提供了类型 `void` 的表达式。 未定义 `void` 操作数的值。
