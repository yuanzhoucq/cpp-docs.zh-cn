# <a name="compiler-warning-level-1-c4088"></a>编译器警告 （等级 1） C4088
function： 指针实际参数 number 形参数字中的不匹配  
  
 相应的形式和实际参数具有不同程度的间接性。 实际的参数传递时未经更改。 所调用的函数会将其值解释为指针。
