# <a name="compiler-warning-level-3-c4281"></a>编译器警告（等级3） C4281

"operator->" 通过类型 "type" 发生递归

你的代码允许**操作员 >** 调用其自身。

下面的示例生成 C4281：

```cpp
// C4281.cpp
// compile with: /W3 /WX
struct A;
struct B;
struct C;

struct A
{
   int z;
   B& operator->();
};

struct B
{
   C& operator->();
};

struct C
{
   A& operator->();
};

void f(A p)
{
   int i = p->z; // C4281
}
```
