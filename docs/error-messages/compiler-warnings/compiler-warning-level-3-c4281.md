# <a name="compiler-warning-level-3-c4281"></a>编译器警告（等级 3）C4281
operator-> 递归发生通过类型 type  
  
 你的代码允许**-> 运算符**调用自身。  
  
 下面的示例生成 C4281:  
  
```  
// C4281.cpp  
// compile with: /W3 /WX  
struct A;  
struct B;  
struct C;  
  
struct A  
{  
   int z;  
   B& operator->();  
};  
  
struct B  
{  
   C& operator->();  
};  
  
struct C  
{  
   A& operator->();  
};  
  
void f(A p)  
{  
   int i = p->z; // C4281  
}  
```
