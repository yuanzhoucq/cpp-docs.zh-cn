# <a name="expression-evaluator-error-cxx0045"></a>表达式计算器错误 CXX0045
不是函数  
  
 不是函数的名称的程序中的符号为提供的自变量列表。  
  
## <a name="example"></a>示例  
  
```  
queue( alpha, beta )  
```  
  
 当`queue`不是函数。  
  
 此错误是与 CAN0045 相同。
