# <a name="expression-evaluator-error-cxx0023"></a>表达式计算器错误 CXX0023

错误的基数

C 表达式计算器无法识别指定的基数。 仅当十进制、 十六进制和八进制 radixes 才有效。

此错误是与 CAN0023 相同。
