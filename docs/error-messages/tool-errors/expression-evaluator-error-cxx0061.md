# 表达式计算器错误 CXX0061
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

超出了函数嵌套深度  
  
 表达式包含嵌套深度超出限制的函数。  修改表达式以减少嵌套深度。  
  
 该错误与 CAN0061 相同。
