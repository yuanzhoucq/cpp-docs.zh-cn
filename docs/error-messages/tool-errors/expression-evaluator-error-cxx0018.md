# 表达式计算器错误 CXX0018
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

错误的寄存器名  
  
 指定寄存器不存在或无法显示。  
  
 “监视”窗口可以显示下列寄存器：  
  
 AX    SP    DS    IP  
  
 BX    BP    ES    FL  
  
 CX    SI    SS    GS  
  
 DX    DI    CS    SS  
  
 EAX    ESP    DS    EIP  
  
 EBX    EBP    ES    EFL  
  
 ECX    ESI    FS  
  
 EDX    EDI    CS  
  
 该错误与 CAN0018 相同。
