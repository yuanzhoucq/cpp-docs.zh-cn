# 资源编译器错误 RC2107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**应输入数字命令值**  
  
 资源编译器希望在 **ACCELERATORS** 语句中有一个数字 *idvalue* 字段。  确保您已经使用 `#define` 常数指定了该值并且正确拼写了该常数。
