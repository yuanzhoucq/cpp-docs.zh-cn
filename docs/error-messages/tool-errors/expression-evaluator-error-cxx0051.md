# <a name="expression-evaluator-error-cxx0051"></a>表达式计算器错误 CXX0051
在断点中指定的多个重载的符号  
  
 表达式计算器无法确定其要用作断点的多个重载符号。  
  
 此错误是与 CAN0051 相同。
