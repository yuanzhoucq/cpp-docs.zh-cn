# 资源编译器错误 RC1047
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“太多的 option 选项，‘string'”  
  
 指定给定选项的次数太多。  给定字符串是导致该错误的选项的参数。
