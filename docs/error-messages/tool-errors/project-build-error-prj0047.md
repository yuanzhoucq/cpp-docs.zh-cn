# <a name="project-build-error-prj0047"></a>项目生成错误 PRJ0047

无法恢复挂起的进程。  生成失败。

开发环境出错。

若要解决此错误，请关闭并重新加载项目。 如有必要，请退出并重新启动 Visual Studio。
