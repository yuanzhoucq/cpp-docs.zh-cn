# <a name="resource-compiler-fatal-error-rc1101"></a>资源编译器错误 RC1101
未指定的资源二进制文件名  
  
 重命名输出 (/ fo) 选项后面不是文件名。  
  
 /Fo 选项使用以下语法：  
  
 **RC /fo**   
 ***outfilename infilename。* rc**
