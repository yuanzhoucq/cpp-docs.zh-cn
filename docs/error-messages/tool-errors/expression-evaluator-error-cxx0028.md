# 表达式计算器错误 CXX0028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

非结构\/联合元素  
  
 已指定“Struct.Member”或“pStruct\-\>Member”形式的表达式，但 \<member\> 不是该结构的元素。  
  
 表达式可能未正确加括号。  
  
 该错误与 CAN0028 相同。
