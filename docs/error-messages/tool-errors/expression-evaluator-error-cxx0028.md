# <a name="expression-evaluator-error-cxx0028"></a>表达式计算器错误 CXX0028
不结构/联合元素  
  
 指定窗体 Struct.Member 或 pstruct->member 的表达式，但\<成员 > 不是元素的结构。  
  
 该表达式可能不正确是用圆括号括起来。  
  
 此错误是与 CAN0028 相同。
