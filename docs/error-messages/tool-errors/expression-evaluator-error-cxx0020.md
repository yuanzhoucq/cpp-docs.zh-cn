# <a name="expression-evaluator-error-cxx0020"></a>表达式计算器错误 CXX0020
此操作的操作数类型错误  
  
 为该运算符时，运算符被应用到具有无效类型的表达式。  
  
 例如，它不是有效才能注册或用浮点表达式作为数组下标的地址的。  
  
 此错误是与 CAN0020 相同。
