# <a name="linker-tools-error-lnk1241"></a>链接器工具错误 LNK1241

资源文件资源文件已指定

如果您运行，则会生成此错误**cvtres**手动从命令行，如果您然后将传递生成的.obj 文件链接器此外到其他.res 文件。

若要指定多个.res 文件，将其传递所有为.res 文件作为链接器不在.obj 文件创建由**cvtres**。
