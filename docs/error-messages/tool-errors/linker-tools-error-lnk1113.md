# <a name="linker-tools-error-lnk1113"></a>链接器工具错误 LNK1113
无效的计算机类型  
  
 对象标头中指定的计算机类型不是有效的。  
  
 如果该文件已损坏，则可以导致此错误。 重新生成对象。
