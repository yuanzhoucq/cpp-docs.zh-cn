# <a name="resource-compiler-error-rc2164"></a>资源编译器错误 RC2164

RCDATA 中的意外的值

*RCDATA* 语句中的 **原始数据** 值必须是整数或字符串，每个值用逗号分隔。 请确保未遗漏逗号或字符串周围的引号。
