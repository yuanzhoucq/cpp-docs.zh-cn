# <a name="resource-compiler-fatal-error-rc1022"></a>资源编译器错误 RC1022

应输入 "#endif"

`#if`、 **#ifdef**或 **#ifndef**指令未以 `#endif` 指令结束。

请确保在此语句之前有 `#if`、 **#ifdef**或 **#ifndef**语句生效。
