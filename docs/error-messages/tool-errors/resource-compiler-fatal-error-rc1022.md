# <a name="resource-compiler-fatal-error-rc1022"></a>资源编译器错误 RC1022

预期 #endif

`#if`， **#ifdef**，或 **#ifndef**指令未终止与`#endif`指令。

请确保是否有`#if`， **#ifdef**，或 **#ifndef**实际上此语句前的语句。
