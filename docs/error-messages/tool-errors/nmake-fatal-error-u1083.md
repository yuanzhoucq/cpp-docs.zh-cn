# NMAKE 错误 U1083
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

目标宏“target”扩展到 nothing  
  
 给定的目标是尚未定义或具有空值的宏的调用。  NMAKE 无法处理空目标。
