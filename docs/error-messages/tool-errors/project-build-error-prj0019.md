# <a name="project-build-error-prj0019"></a>项目生成错误 PRJ0019

工具返回了错误代码

自定义生成步骤或生成事件的错误级别不为零。

当工具返回了错误代码，但没有出现错误消息时，还会看到 PRJ0019。 例如，如果将 MIDL 的输出重定向到 NUL，就会发生这种情况。

有关详细信息，请参阅[自定义生成步骤疑难解答和生成事件](../../build/troubleshooting-build-customizations.md)。

当你作为用户组的成员运行并且需要管理访问权限时，也会发生此错误。 有关详细信息，请参阅[以用户组成员的身份运行](../../security/running-as-a-member-of-the-users-group.md)。
