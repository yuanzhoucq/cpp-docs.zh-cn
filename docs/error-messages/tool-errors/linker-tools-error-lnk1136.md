# <a name="linker-tools-error-lnk1136"></a>链接器工具错误 LNK1136
无效或已损坏的文件  
  
 输入的文件已损坏的标头，或为零大小或异常小。  
  
 如果文件不正确地指定为附加依赖项，可能还会导致 LNK1136。
