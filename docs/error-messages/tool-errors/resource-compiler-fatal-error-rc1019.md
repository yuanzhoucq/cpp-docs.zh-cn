# <a name="resource-compiler-fatal-error-rc1019"></a>资源编译器错误 RC1019
意外 #else  
  
 `#else`指令未出现在`#if`， **#ifdef**，或 **#ifndef**构造。  
  
 请确保有`#if`， **#ifdef**，或 **#ifndef**实际上此语句前的语句。
