# <a name="resource-compiler-fatal-error-rc1019"></a>资源编译器错误 RC1019

意外 #else

`#else`中不显示指令`#if`， **#ifdef**，或 **#ifndef**构造。

请确保是否有`#if`， **#ifdef**，或 **#ifndef**实际上此语句前的语句。
