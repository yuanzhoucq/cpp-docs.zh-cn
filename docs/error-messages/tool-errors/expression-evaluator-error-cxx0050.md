# <a name="expression-evaluator-error-cxx0050"></a>表达式计算器错误 CXX0050
左侧的:: 必须是类/结构/联合  
  
 范围解析运算符左侧的符号 (`::`) 不是类、 结构或联合。  
  
 此错误是与 CAN0050 相同。
