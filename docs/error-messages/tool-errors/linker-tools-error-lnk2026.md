# <a name="linker-tools-error-lnk2026"></a>链接器工具错误 LNK2026

模块对于 SAFESEH 映像是不安全

[/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md)已指定，但模块不兼容与安全异常处理功能。 如果你想要使用与此模块 **/SAFESEH**，则您将需要重新编译该模块。
