# 链接器工具警告 LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

找到无效的指令 \/directive；已将其忽略  
  
 通过 [comment](../../preprocessor/comment-c-cpp.md) 杂注传递的指令无效。  链接器忽略 `/directive`。
