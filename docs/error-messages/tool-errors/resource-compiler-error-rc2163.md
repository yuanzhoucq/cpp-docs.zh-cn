# 资源编译器错误 RC2163
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

需要快捷键类型 \[ASCII 或 VIRTKEY\]  
  
 **ACCELERATORS** 语句中的 `type` 字段必须包含 ASCII 或 VIRTKEY 值。
