# <a name="resource-compiler-error-rc2163"></a>资源编译器错误 RC2163
加速器类型所需 [ASCII 或 VIRTKEY]  
  
 `type` ACCELERATORS **语句中的** 字段中必须包含 ASCII 或 VIRTKEY 值。
