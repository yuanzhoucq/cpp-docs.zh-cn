# 表达式计算器错误 CXX0053
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

绑定断点时无函数符号匹配  
  
 用作断点的符号不是一个函数。  将数据成员指定为断点可导致该错误。  
  
 该错误与 CAN0053 相同。
