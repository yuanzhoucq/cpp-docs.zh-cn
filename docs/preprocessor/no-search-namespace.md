# <a name="nosearchnamespace"></a>no_search_namespace
**C + + 专用**  
  
 具有相同的功能[no_namespace](../preprocessor/no-namespace.md)属性，但在使用与 #import 指令的类型库上使用[auto_search](../preprocessor/auto-search.md)属性。  
  
## <a name="syntax"></a>语法  
  
```  
no_search_namespace  
```  
  
## <a name="remarks"></a>备注  
 **结束 c + + 专用**
