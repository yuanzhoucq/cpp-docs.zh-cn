# <a name="c6281"></a>C6281

> 警告 C6281-操作顺序不正确：关系运算符的优先级高于位运算符

此警告表明运算符优先级可能存在错误。 这可能会产生不正确的结果。 你应检查优先级，并使用括号来阐明这一目的。 关系运算符（\<、>、\<=、> =、= =、！ =）的优先级高于位运算符（& &#124; ^）。

## <a name="example"></a>示例

下面的代码将生成此警告：

```cpp
#include <stdlib.h>
#define FORMAT 1
#define TYPE 2

void f(int input)
{
  if (FORMAT & TYPE != input)
  {
    // code...
  }
}
```

下面的代码使用括号来更正此警告：

```cpp
#include <stdlib.h>
#define FORMAT 1
#define TYPE 2

void f(int input)
{
  if ((FORMAT & TYPE) != input)
  {
    // code...
  }
}
```
