# <a name="c28236"></a>C28236

> 警告 C28236：不能在表达式中使用批注

此警告意味着尝试使用声明为表达式上下文中的批注的函数。 如果正对批注使用标准宏，则这应不是可能实现的；此警告在正常使用时应不显示。
