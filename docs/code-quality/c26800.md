# <a name="c26800"></a>C26800

> 警告 C26800：使用对象的已移动对象： \<锁定 >。

当在从中移动变量后使用该变量时，将触发 C26800 警告。 在将变量作为 rvalue 引用传递到函数后，将其视为已从移动。 一些用于赋值、析构和某些状态重置功能的合法异常，如 std：： vector：： clear。

## <a name="example"></a>示例

下面的代码将生成 C26800。

```cpp

#include <utility>

struct X {
    X();
    X(const X&);
    X(X&&);
    X &operator=(X&);
    X &operator=(X&&);
    ~X();
};

template<typename T>
void use_cref(const T&);

void test() {
    X x1;
    X x2 = std::move(x1);
    use_cref(x1);                // @expected(26800)
}
```

## <a name="example"></a>示例

下面的代码不会生成 C26800。

```cpp

#include <utility>

struct MoveOnly {
    MoveOnly();
    MoveOnly(MoveOnly&) = delete;
    MoveOnly(MoveOnly&&);
    MoveOnly &operator=(MoveOnly&) = delete;
    MoveOnly &operator=(MoveOnly&&);
    ~MoveOnly();
};

template<typename T>
void use(T);

void test() {
    MoveOnly x;
    use(std::move(x)); // no 26800
}
```
