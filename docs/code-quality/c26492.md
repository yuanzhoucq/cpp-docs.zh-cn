# <a name="c26492-no_const_cast"></a>C26492 NO_CONST_CAST

不要使用 `const_cast` 来强制转换为 `const`。 请参阅[ C++核心准则类型。 3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type)。
