# <a name="c28024"></a>C28024

> 警告 C28024：所分配的函数指针是用函数类（> 类 \<类）批注的，该函数未包含在函数类中 \<t >。

当两个函数使用一个函数类进行批注，但类不匹配时，将报告此警告。

当函数指针具有 `_Function_class_` 批注，且该批注规定只应向其分配特定函数类的函数时，将发出此警告。 在函数调用的分配或隐含分配中，源和目标必须是同一函数类，但函数类不匹配。
