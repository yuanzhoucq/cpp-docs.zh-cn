# <a name="c26494-var_use_before_init"></a>C26494 VAR_USE_BEFORE_INIT

变量 "% variable%" 未初始化。 始终初始化对象。 请参阅[ C++核心准则类型 5](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type)。
