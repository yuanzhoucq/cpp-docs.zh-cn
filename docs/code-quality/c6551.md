# <a name="c6551"></a>C6551

> 警告 C6551：大小规范无效：表达式不可分析

> [!NOTE]
> 此警告仅发生在使用源代码批注语言 (SAL) 已弃用版本的代码中。 建议你移植代码以使用 SAL 的最新版本。 有关详细信息，请参阅[使用 SAL 注释减少 C/C++代码缺陷](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md)。
