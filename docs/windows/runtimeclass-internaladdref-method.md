# <a name="runtimeclassinternaladdref-method"></a>RuntimeClass::InternalAddRef 方法
递增对当前 RuntimeClass 对象的引用计数。  
  
## <a name="syntax"></a>语法  
  
```  
ULONG InternalAddRef();  
```  
  
## <a name="return-value"></a>返回值  
 在生成的引用计数。  
  
## <a name="requirements"></a>要求  
 **标头：** implements.h  
  
 **命名空间：** Microsoft::WRL
