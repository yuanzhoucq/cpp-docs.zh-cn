# <a name="comptrcomptr-destructor"></a>ComPtr::~ComPtr 析构函数
取消初始化 ComPtr 的实例。  
  
## <a name="syntax"></a>语法  
  
```  
WRL_NOTHROW ~ComPtr();  
```  
  
## <a name="requirements"></a>要求  
 **标头：** client.h  
  
 **命名空间：** Microsoft::WRL
