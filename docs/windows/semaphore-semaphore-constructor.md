# <a name="semaphoresemaphore-constructor"></a>Semaphore::Semaphore 构造函数

## <a name="syntax"></a>语法  
  
```  
explicit Semaphore(  
   HANDLE h  
);  
  
WRL_NOTHROW Semaphore(  
   _Inout_ Semaphore&& h  
);  
```  
  
### <a name="parameters"></a>参数  
 *h*  
 句柄或对右值引用**信号量**对象。
