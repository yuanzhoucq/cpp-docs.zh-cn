# <a name="ptr"></a>ptr

将指针指定为完全指针。

## <a name="syntax"></a>语法

```cpp
[ptr]
```

## <a name="remarks"></a>备注

**Ptr** C++特性具有与[ptr](/windows/win32/Midl/ptr) MIDL 特性相同的功能。

## <a name="example"></a>示例

有关**ptr**的示例用法, 请参阅[defaultvalue](defaultvalue.md)的示例。
