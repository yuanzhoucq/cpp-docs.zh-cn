# <a name="switch_is"></a>switch_is

指定充当用于选择联合成员的联合判别的表达式或标识符。

## <a name="syntax"></a>语法

```cpp
[switch_is]
```

## <a name="remarks"></a>备注

**Switch_is** C++特性具有与[switch_is](/windows/win32/Midl/switch-is) MIDL 特性相同的功能。
